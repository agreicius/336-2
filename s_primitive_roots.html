    Section 1.3 Primitive roots

    Objectives

    1. Develop some theory about polynomials with coefficients in one of our familiar rings (e.g., \(\Z\text{,}\) \(\Z/n\Z\text{,}\) \(\Q\text{,}\) \(\R\)).
      🔗
      🔗
    2. Prove that \((\Z/p\Z)^*\) has a primitive root for \(p\) prime.
      🔗
      🔗
    3. Derive counting formulas for elements of given order \(k\) in \((\Z/n\Z)^*\text{,}\) assuming that a primitive root exists.
      🔗
      🔗
    🔗

    Subsection 1.3.1 Introduction

    In this section we will show that \((\Z/p\Z)^*\) has a primitive root for \(p\) prime. Before getting to that existential statement, we first prove a conditional result that counts the number of primitive roots modulo \(n\text{,}\) assuming one exists. Note that Theorem 1.3.7 provides an unconditional version of this result in the special case where \(n=p\) is a prime integer.
    🔗

    Theorem 1.3.1. Number of primitive roots.

    Let \(n\) be a positive integer.
    1. If \((\Z/n\Z)^*\) has a primitive root, then it has exactly \(\phi(\phi(n))\) distinct primitive roots.
      🔗
      🔗
    2. More generally, if \((\Z/n\Z)^*\) has a primitive root, then for each positive divisor \(d\) of \(\phi(n)\text{,}\) the number of elements of \((\Z/n\Z)^*\) of order \(d\) is \(\phi(d)\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.

    Assume \(r\) is a primitive root of \((\Z/n\Z)^*\text{,}\) so that
    \begin{align*} (\Z/n\Z)^* \amp = \{1,r,r^2,\dots, r^{\phi(n)-1}\}\text{.} \end{align*}
    1. If \(s\) is a primitive root of \((\Z/n\Z)^*\text{,}\) then \(s=r^i\) for some \(i\in \{0,1,\dots, \phi(n)-1\}\text{.}\) And furthermore, since \(\ord s=\phi(n)\text{,}\) we must have
      \begin{align*} \phi(n) \amp =\ord s\\ \amp = \ord r^i\\ \amp = \phi(n)/\gcd(i,\phi(n))\text{,} \end{align*}
      by Proposition 1.2.3. This happens if and only if \(\gcd(i,\phi(n))=1\text{.}\) Thus , the number of primitive roots of \((\Z/n\Z)^*\) is equal in this case to the number of \(i\in \{0,1,\dots, \phi(n)-1\}\) satisfying \(\gcd(i,\phi(n))=1\text{.}\) This is \(\phi(\phi(n))\text{,}\) by definition.
      🔗
      🔗
    2. This result is very easy to prove if you know a little group theory, and a little tricky to prove (for general \(n\)) using just elementary number theory. We will defer the proof to a guided homework exercise.
      🔗
      🔗
    🔗
    🔗

    Example 1.3.2. Primitive roots modulo 25.

    We saw in Example 1.2.5 that \((\Z/25\Z)^*\) has a primitive root, and that in fact \(2, 3, 8, 12, 13,17, 22, 23\) are all of the primitive roots of \((\Z/25\Z)^*\text{.}\) Observe that there are exactly \(8\) of these; and sure enough, we have
    \begin{align*} \phi(\phi(25)) \amp =\phi(20)\\ \amp = \phi(4)\phi(5)\\ \amp = 2\cdot 4=8\text{.} \end{align*}
    🔗
    🔗
    We now make a small digression to develop some notions about polynomials with coefficients in a given number system. We could develop this theory for polynomials with coefficients in an arbitrary ring \(R\text{,}\) but since we have not given an official definition of a ring, we will restrict ourselves to the case where our coefficients lie in \(\Z\text{,}\) \(\Q\text{,}\) \(\R\text{,}\) or \(\Z/n\Z\) for some positive integer \(n\text{.}\)
    🔗
    🔗

    Subsection 1.3.2 Polynomials with coefficients in rings

    Definition 1.3.3. Polynomials over a ring.

    Let \(R\) be either \(\Z\text{,}\) \(\Q\text{,}\) \(\R\text{,}\) or \(\Z/n\Z\) for some positive integer \(n\text{.}\) A polynomial over \(R\) in the indeterminate (or variable) \(x\) is a formal expression of the form
    \begin{align*} f(x) \amp =a_nx^n+a_{n-1}x^{n-1}+\cdots a_1x+a_0\text{,} \end{align*}
    where \(n\) is a nonnegative integer and \(a_i\in R\) for all \(0\leq i\leq n\text{.}\) We call \(a_i\) the coefficients of \(f\text{,}\) and we denote by \(R[x]\) the set of all polynomials with coefficients in \(R\text{:}\) i.e.,
    \begin{align} R[x] \amp =\{\anpoly\mid n\in \Z_{\geq 0}, a_i\in R\}\text{.}\tag{1.3.1} \end{align}
    The polynomial is nonzero if \(a_i\ne 0\) for some \(i\text{,}\) and zero otherwise. The degree of a nonzero polynomial \(f\text{,}\) denoted \(\deg f\text{,}\) is the largest positive integer \(n\) such that the \(n\)-th coefficient of \(f\) is nonzero.
    🔗
    Two nonzero polynomial \(f(x)=\anpoly\) and \(g(x)=\bmpoly\) with \(a_n\ne 0\) and \(b_m\ne 0\) are equal, written \(f=g\) if \(n=m\) and \(a_i=b_i\) for all \(0\leq i\leq n\text{.}\)
    🔗
    Given a polynomial \(f(x)=\anpoly\) and an element \(r\in F\text{,}\) the evaluation of \(f\) at \(r\) is the element \(f(r)\) defined as
    \begin{align} f(r) \amp = a_nr^n+a_{n-1}r^{n-1}+\cdots a_1r+a_0 \text{.}\tag{1.3.2} \end{align}
    In this manner, the polynomial \(f\) defines a function \(f\colon R\rightarrow R\text{.}\) An element \(r\in F\) is a root of \(f\) if \(f(r)=0\text{.}\)
    🔗
    Lastly, addition and multiplication of polynomials are defined as usual. In particular, given \(f(x)=\anpoly\) and \(g(x)=\bmpoly\text{,}\) we have
    \begin{align} f(x)g(x) \amp =c_{n+m}x^{n+m}+\cdots +c_1x+c_0\text{,}\tag{1.3.3} \end{align}
    where
    \begin{align} c_k \amp =\sum_{i=0}^ka_ib_{k-i}\tag{1.3.4} \end{align}
    for all \(0\leq k\leq n+m\text{.}\)
    🔗
    🔗
    Although we stated our definitions for polynomials over general rings, for the next theorem we need to restrict ourselves to the case where the coefficients lie in a field. Again, since we have not given a definition of a field, we will state the results for polynomials with coeffiecients in \(\Q\text{,}\) \(\R\text{,}\) or \(\F_p\) for some prime integer \(p\text{.}\)
    🔗

    Theorem 1.3.4. Polynomial properties.

    Let \(F\) be either \(\Q\text{,}\) \(\R\text{,}\) or \(\F_p\) for some prime integer \(p\text{,}\) and let \(f\) and \(g\) be nonzero polynomials over \(F\text{.}\)
    1. Degree of products.
      \(\deg fg=\deg f+\deg g\text{.}\)
      🔗
      🔗
    2. Division with remainder.
      There are unique polynomials \(q\) and \(r\) such that \(f(x)=q(x)g(x)+r(x)\) and either \(r(x)=0\) or \(0\leq \deg r< \deg g \text{.}\)
      🔗
      🔗
    3. Roots and factorization.
      An element \(r\in F\) is a root of \(f\) if and only if we have \(f(x)=(x-r)g(x)\) for some polynomial \(g\text{.}\)
      🔗
      🔗
    4. Roots and degree.
      Let \(\deg f=n\text{.}\) There are at most \(n\) distinct roots of \(f\) in \(F\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.

    In the arguments below, besides the ring structure of \(F\text{,}\) we will use two crucial additional properties that are related to one another:
    • If \(r\in F\) is nonzero, then there is an element \(r^{-1}\) such that \(rr^{-1}=1\text{.}\)
      🔗
      🔗
    • If \(r\) and \(s\) are nonzero elements of \(F\text{,}\) then \(rs\) is nonzero.
      🔗
      🔗
    The second property is actually a consequence of the first: assume \(r\) is nonzero; by the first property, there is an element \(r^{-1}\) satisfying \(r^{-1}r=1\text{;}\) if \(rs=0\) for some element \(s\in F\text{,}\) then multiplying both sides by \(r^{-1}\text{,}\) we conclude that \(s=0\text{;}\) thus if \(r\) and \(s\) are nonzero, then \(rs\) is nonzero.
    🔗
    Now for the proofs of the four statements.
    1. Let \(f(x)=\anpoly\) and \(g(x)=\bmpoly\text{,}\) where \(a_n\ne 0\) and \(b_m\ne 0\text{,}\) so that \(\deg f=n\) and \(\deg g=m\text{.}\) We then have
      \begin{align} f(x)g(x) \amp = a_nb_mx^{n+m}+\text{lower order terms}\text{.}\tag{1.3.5} \end{align}
      Since \(a_n\) and \(b_m\) are nonzero, then \(a_nb_m\ne 0\) (See above.) It follows that \(\deg fg=n+m=\deg f+\deg g\text{.}\)
      🔗
      🔗
    2. We omit a formal proof of this result, which is basically a consequence of the fact that for any of the given number systems \(F\) we can perform polynomial long division in the same manner as we do over \(\R\text{.}\)
      🔗
      🔗
    3. For any \(r\in F\text{,}\) using division with remainder, we can write \(f\) uniquely as
      \begin{align*} f(x) \amp =q(x)(x-r)+s(x) \end{align*}
      where either \(s(x)=0\) or \(0\leq \deg s(x)< \deg (x-r)=1\text{.}\) In other words, we have \(f(x)=q(x)(x-r)+c\) for some constant \(c\in F\text{.}\) Since \(f(r)=q(r)(r-r)+c=c\text{,}\) we see that \(r\) is a root of \(f\) if and only if \(c=0\text{,}\) if and only if \(f(x)=q(x)(x-r)\text{.}\)
      🔗
      🔗
    4. We proceed by induction on \(n=\deg f\text{.}\)
      🔗
      Base step: \(n=0\text{.}\) Since \(\deg f=0\) in this case, we have \(f(x)=c\) for some constant \(c\in F\text{.}\) Since \(f\) is assumed to be nonzero, we have \(c\ne 0\text{.}\) But then \(f(r)=c\ne 0\) for all \(r\in F\text{,}\) showing that \(f\) has zero roots.
      🔗
      🔗
    5. Let \(n\geq 0\text{,}\) and assume the result holds for all nonzero polynomials of degree \(n\text{.}\) Let \(f\) be a polynomial of degree \(n+1\text{.}\) If \(f\) has no roots, then we are done. Otherwise, let \(r\in F\) be a root of \(f\text{,}\) and write \(f(x)=(x-r)g(x)\text{.}\) Since \(n+1=\deg f=\deg(x-r)+\deg g(x)=1+\deg g(x)\text{,}\) we see that \(\deg g=n\) (and in particular, \(g\) is nonzero). Suppose \(s\ne r\) is another root of \(f\text{.}\) Since \(f(s)=(s-r)g(s)=0\) and \(s-r\ne 0\text{,}\) we must have \(g(s)=0\text{.}\) (Again, see the first paragraph of this proof.) Thus any other root of \(f\) besides \(r\) is a root of \(g\text{.}\) Since \(g\) has degree \(n\text{,}\) it has at most \(n\) distinct roots by induction. We conclude that \(f\) has at most \(n+1\) distinct roots, as desired.
      🔗
      🔗
    🔗
    🔗
    🔗

    Subsection 1.3.3 Primitive roots modulo a prime

    With this bit of polynomial theory under our belt, we move on to a proof that \((\Z/p\Z)^*\) has a primitive root for \(p\) prime. We begin with two lemmas. The first is a classic identity involving the Euler totient function that is usually covered in the first quarter of the class.
    🔗

    Lemma 1.3.5. Totient function identity.

    Let \(n\) be a positive integer, and let \(\phi\) be the Euler totient function.
    \begin{align} \sum_{d\mid n}\phi(d) \amp = n \text{.}\tag{1.3.6} \end{align}
    🔗
    🔗

    Lemma 1.3.6. Elements of order \(k\) bound.

    Let \(p\) be a prime integer, and let \(k\) be a positive integer. The number of elements of \((\Z/p\Z)^*\) of order \(k\) is either \(0\) or \(\phi(k)\text{.}\)
    🔗
    🔗

    Proof.

    If there are no elements of \((\Z/p\Z)^*\) of order \(k\text{,}\) then we are done. Otherwise, let \(r\in (\Z/p\Z)^*\) have order \(k\text{.}\) It follows that each of the \(k\) elements \(x=r^i\in \angvec{r}\) satisfies \(x^k=1\text{,}\) or equivalently \(x^k-1=0\text{.}\) Since the polynomial \(f(x)=x^k-1\) has coefficients in the field \(\F_p=\Z/p\Z\text{,}\) and has degree equal to \(k\text{,}\) \(f\) has at most \(k\) roots. We conclude that the roots of \(f\) are precisely the elements of \(\angvec{r}\text{.}\) In particular, any other element \(s\) of order \(k\) must be of the form \(s=r^i\) for some \(i\in \{0,1,\dots, k-1\}\text{.}\) Since \(\ord r^i=k/\gcd(i,k)\text{,}\) we see that \(s=r^i\) has order \(k\) if and only if \(\gcd(i,k)=1\text{.}\) We conclude that there are precisely \(\phi(k)\) elements of \((\Z/p\Z)^*\) of order \(k\text{.}\)
    🔗
    🔗

    Theorem 1.3.7. Primitive roots modulo \(p\).

    Let \(p\) be a prime integer. For each positive integer \(k\text{,}\) let \(F_k\) be the set of elements of \((\Z/p\Z)^*\) of order \(k\text{.}\)
    1. If \(k\nmid \phi(p)=p-1\text{,}\) then \(F_k=\emptyset\text{.}\)
      🔗
      🔗
    2. If \(k\mid \phi(p)\text{,}\) then \(\abs{F_k}=\phi(k)\text{.}\)
      🔗
      🔗
    3. In particular, \(\abs{F_{\phi(p)}}=\phi(p-1)\ne 0\text{,}\) and thus there is a primitive root \(r\) of \(\Z/p\Z\text{:}\) i.e., there is an element \(r\in (\Z/p\Z)^*\) satisfying \(\angvec{r}=(\Z/p\Z)^*\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.

    1. We know from Proposition 1.2.3 that \(\ord r\mid \phi(p)\) for all \(p\in \F_p^*\text{.}\) Thus, if \(k\nmid \phi(p)\text{,}\) then there is no element \(r\in (\Z/p\Z)^*\) of order \(k\text{.}\) Equivalently, \(F_k=\nonempty\) for all \(k\nmid \phi(p)\text{.}\)
      🔗
      🔗
    2. For each positive divisor \(d\) of \(\phi(p)\) we know from Lemma 1.3.6 that
      \begin{align*} \abs{F_d} \amp \in \{0,\phi(d)\}\text{.} \end{align*}
      In particular, we have \(\abs{F_d}\leq \phi(d)\) for all \(d\mid \phi(p)\text{.}\) Furthermore, since the sets \(F_d\) are disjoint (an element of \((\Z/p\Z)^*\) has a unique order), and since \((\Z/p\Z)^*=\bigcup_{d\mid n}F_d\) (every element of \((\Z/p\Z)^*\) has order \(d\) for some divisor \(d\) of \(\phi(p)\)), we have
      \begin{align} p-1 \amp =\abs{\F_p^*}=\sum_{d\mid p-1}\abs{F_d} \text{.}\tag{1.3.7} \end{align}
      From Lemma 1.3.5 we also know that
      \begin{align*} p-1 \amp = \sum_{d\mid p-1}\phi(d)\text{.} \end{align*}
      Subtracting these two equalities, we conclude that
      \begin{align*} 0 \amp =\sum_{d\mid p-1}(\phi(d)-\abs{F_d})\text{.} \end{align*}
      Since \(\abs{F_d}\leq \phi(d)\) for all positive \(d\mid p-1\text{,}\) we conclude that \(\phi(d)-\abs{F_d}\geq 0\) for all positive \(d\mid p-1\text{.}\) But a sum of nonnegative numbers is zero if and only if each number is zero. Thus we conclude that
      \begin{align*} \abs{F_d} \amp = \phi(d) \end{align*}
      for all positive \(d\mid p-1\text{.}\)
      🔗
      🔗
    3. From (2), we conclude that \(\ans{F_{p-1}}=\phi(p-1)\text{.}\) Since \(\phi(p-1)\) is positive, we conclude that there is an element \(r\in \F_p^*\) of order \(p-1\text{.}\) (In fact there are precisely \(\phi(p-1)\) such elements.) Such an element satisfies \(\angvec{r}=\F_p^*\) and hence is a primitive root.
      🔗
      🔗
    🔗

    Sage example 3. Primitive roots.

    The Sage function primitive_root(n) returns a primitive root of \((\Z/n\Z)^*\) if one exists, and raises an error otherwise. Furthermore, if \(n=p\) is prime, then primitive_root(p) returns the smallest positive integer \(a\) such that \([a]_p\) is a primitive root of \((\Z/p\Z)^*\text{.}\)
