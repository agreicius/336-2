Theorem 1.15.11. Algebraic integers form a ring.

Let \(\mathcal{O}\) be the set of all algebraic integers in \(\C\text{:}\) i.e.,
\begin{equation*} \mathcal{O}=\{\alpha\in \C\mid f(\alpha)=0 \text{ for some monic } f\in \Z[x]\}\text{.} \end{equation*}
  1. \(\mathcal{O}\cap \Q=\Z\text{.}\) In other words, all integers are algebraic integers, and furthermore, the set of rational numbers that are algebraic integers is precisely \(\Z\text{.}\)
    🔗
    🔗
  2. \(\mathcal{O}\) is closed under addition and multiplication: i.e., we have
    \begin{align*} \alpha,\beta\in \mathcal{O} \amp \implies \alpha+\beta, \alpha\beta\in \mathcal{O}\text{.} \end{align*}
    🔗
    🔗
As a result, \(\mathcal{O}\) is a subring of \(\C\text{.}\)
🔗
🔗

Proof.

  1. If \(n\in \Z\text{,}\) then \(f(n)=0\) for the monic integer polynomial \(f(x)=x-n\text{.}\) Thus \(\Z\subseteq \mathcal{O}\cap \Q\text{.}\)
    🔗
    Conversely, suppose \(q\in \mathcal{O}\cap \Q\text{.}\) Let \(f(x)=\anmonic\) be a monic polynomial satisfying \(f(q)=0\text{.}\) Writing \(q=r/s\) with \(\gcd(r,s)=1\) and clearing denominators in the equation \(f(r/s)=0\text{,}\) we see that
    \begin{equation*} r^n+a_{n-1}r^{n-1}s+\cdots +a_1rs^{n-1}+a_0s^n=0\text{.} \end{equation*}
    Since all constants above are integers, we see that \(s\mid r^n\text{,}\) a contradiction since \(\gcd(r,s)=1\text{.}\)
    🔗
    🔗
  2. As intuitive as this property might sound, we resort to a little linear algebra trickery to prove it. Namely, we first prove the following sufficient condition for an element \(\alpha\in \C\) to be an algebraic integer: if there is a set \(\{\theta_1,\theta_2,\dots, \theta_k\}\) of complex numbers, not all zero, such that for all \(1\leq j\leq k\) we have
    \begin{equation*} \alpha\theta_j=\sum_{i=1}^kb_{ij}\theta_i \end{equation*}
    for some integer constants \(b_{ij}\in \Z\text{,}\) then \(\alpha\) is an algebraic integer.
    🔗
    Before proving this claim, let’s see why it yields the desired result. Suppose that \(\alpha\) and \(\beta\) are algebraic integers satisfying \(f(\alpha)=0\) and \(g(\beta)=0\text{,}\) where \(f(x)=\anmonic\) and \(g(x)=x^m+b_{m-1}x^{m-1}+\cdots +b_1x+b_0\) are monic integer polynomials. Since
    \begin{align*} \alpha^n \amp =-\sum_{i=0}^{n-1}a_i\alpha^{i}\\ \beta^m \amp = -\sum_{i=0}^{m-1}b_i\beta^{i}\text{,} \end{align*}
    it follows that the set
    \begin{equation*} X=\{\alpha^{i}\beta^{j}\mid 0\leq i\leq n-1, 0\leq j\leq m-1\} \end{equation*}
    satisfies the condition above for \(\alpha\) and \(\beta\text{.}\) Indeed, we have
    \begin{align*} \alpha (\alpha^i\beta^j) \amp = \begin{cases} \alpha^{i+1}\beta^j\amp \text{if } 0\leq i\leq n-2\\ -\sum_{\ell=0}^{n-1}a_{\ell}\alpha^\ell \beta^j\amp \text{if } i=n-1 \end{cases}\\ \beta (\alpha^i\beta^j) \amp = \begin{cases} \alpha^{i}\beta^{j+1}\amp \text{if } 0\leq j\leq m-2\\ -\sum_{\ell=0}^{m-1}b_{\ell}\alpha^i \beta^\ell \amp \text{if } j=m-1 \text{.}\end{cases} \end{align*}
    But then it follows easily that \(\alpha+\beta\) and \(\alpha\beta\) also satisfy the claims condition with respect to \(X\text{,}\) showing that these elements are also algebraic integers.
