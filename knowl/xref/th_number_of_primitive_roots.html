Theorem 1.3.1. Number of primitive roots.

Let \(n\) be a positive integer.
  1. If \((\Z/n\Z)^*\) has a primitive root, then it has exactly \(\phi(\phi(n))\) distinct primitive roots.
    🔗
    🔗
  2. More generally, if \((\Z/n\Z)^*\) has a primitive root, then for each positive divisor \(d\) of \(\phi(n)\text{,}\) the number of elements of \((\Z/n\Z)^*\) of order \(d\) is \(\phi(d)\text{.}\)
    🔗
    🔗
🔗
🔗

Proof.

Assume \(r\) is a primitive root of \((\Z/n\Z)^*\text{,}\) so that
\begin{align*} (\Z/n\Z)^* \amp = \{1,r,r^2,\dots, r^{\phi(n)-1}\}\text{.} \end{align*}
  1. If \(s\) is a primitive root of \((\Z/n\Z)^*\text{,}\) then \(s=r^i\) for some \(i\in \{0,1,\dots, \phi(n)-1\}\text{.}\) And furthermore, since \(\ord s=\phi(n)\text{,}\) we must have
    \begin{align*} \phi(n) \amp =\ord s\\ \amp = \ord r^i\\ \amp = \phi(n)/\gcd(i,\phi(n))\text{,} \end{align*}
    by Proposition 1.2.3. This happens if and only if \(\gcd(i,\phi(n))=1\text{.}\) Thus , the number of primitive roots of \((\Z/n\Z)^*\) is equal in this case to the number of \(i\in \{0,1,\dots, \phi(n)-1\}\) satisfying \(\gcd(i,\phi(n))=1\text{.}\) This is \(\phi(\phi(n))\text{,}\) by definition.
    🔗
    🔗
  2. This result is very easy to prove if you know a little group theory, and a little tricky to prove (for general \(n\)) using just elementary number theory. We will defer the proof to a guided homework exercise.
