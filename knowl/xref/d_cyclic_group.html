Definition 1.2.4. Primitive root.

Let \(n\) be a positive integer, and let \(r\in (\Z/n\Z)^*\text{.}\) The cyclic group generated by \(r\), denoted \(\langle r\rangle\text{,}\) is the set of all integer powers of \(r\text{:}\) i.e.,
\begin{align*} \angvec{r} \amp = \{r^i\mid i\in \Z\}\\ \amp = \{1,r, \dots, r^{m-1}\}\text{,} \end{align*}
where \(m=\ord r\text{.}\)
🔗
A primitive root (or cyclic generator) of \((\Z/n\Z)^*\) is an element \(r\in (\Z/n\Z)^*\) satisfying \(\angvec{r}=(\Z/n\Z)^*\text{.}\)
🔗
We say an integer \(a\) is a primitive root modulo \(n\) if \(r=[a]_n\) is a primitive root of \((\Z/n\Z)^*\text{.}\)
