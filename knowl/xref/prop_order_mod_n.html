Proposition 1.2.3. Order of modular units.

Let \(n\) be a positive integer, let \(r\in (\Z/n\Z)^*\text{,}\) and let \(m=\ord r\text{.}\)
  1. For all integers \(k\text{,}\) \(r^k=1\) if and only if \(m\mid k\text{.}\)
    🔗
    🔗
  2. \(r^i=r^j\) if and only if \(i\equiv j\pmod m\text{.}\)
    🔗
    🔗
  3. We have
    \begin{align*} \{r^i\mid i\in \Z\} \amp = \{1,r,r^2,\dots, r^{m-1}\}\text{,} \end{align*}
    and this set has cardinality \(m\) (i.e., it contains exactly \(m\) distinct elements).
    🔗
    🔗
  4. \(\ord r^i=m/\gcd(i,m)\) for all integers \(i\in \Z\text{.}\)
    🔗
    🔗
🔗
🔗

Proof.

  1. First assume that \(m\mid k\text{,}\) so that \(k=mq\) for some \(q\in \Z\text{.}\) We then have
    \begin{align*} r^k \amp =r^{mk}\\ \amp = (r^m)^k\\ \amp =1^k=1\text{,} \end{align*}
    as desired. Conversely, assume \(r^k=1\) and write \(k=qm+s\) where \(0\leq s < m\text{,}\) as in the division algorithm. Since
    \begin{align*} 0 \amp =r^{k}\\ \amp = r^{qm+s}\\ \amp = r^{qm}r^s\\ \amp =r^s \amp (m\mid qm)\text{,} \end{align*}
    and since \(m\) is the smallest positive integer satisfying \(r^m=1\text{,}\) we must have \(s=0\text{,}\) and thus \(m\mid k\text{.}\)
    🔗
    🔗
  2. We have
    \begin{align} r^i=r^j \amp \iff r^{i}r^{-j}=r^{j}r^{-j}\tag{1.2.1}\\ \amp \iff r^{i-j}=1\tag{1.2.2}\\ \amp \iff m\mid (i-j) \amp (\text{by } (1))\tag{1.2.3}\\ \amp \iff i\equiv j\pmod m\text{.}\tag{1.2.4} \end{align}
    🔗
    🔗
  3. Since \(r^i=r^j\) if and only if \(i\equiv j\pmod m\text{,}\) and since \(\{0,1,\dots, m-1\}\) form a complete set of residues modulo \(m\text{,}\) we have
    \begin{align} \{r^i\mid i\in \Z\} \amp = \{r^0, r^1,\dots, r^{m-1}\} \tag{1.2.5}\\ \amp =\{1,r,\dots, r^{m-1}\}\text{,}\tag{1.2.6} \end{align}
    and these powers of \(r\) are all distinct. Thus the set has cardinality \(m\text{.}\)
    🔗
    🔗
  4. Let \(k=m/\gcd(i,m)\text{.}\) We will show that \(k\) is the smallest positive integer such that \((r^i)^k=1\text{,}\) and thus that \(\ord r^i=k\text{.}\)
    🔗
    First observe that \((r^{i})^k=r^{ik}=r^{(i/\gcd(i,m))m}=1\text{,}\) since \(m\mid (i/\gcd(i,m))m\text{.}\) Now assume that \(\ell\) is a positive integer satisfying \((r^i)^\ell=r^{i\ell}=1\text{.}\) It follows that \(m\mid i\ell\text{.}\) But then
    \begin{align*} m/\gcd(i,m) \amp \mid (i/\gcd(i,m))\ell\text{.} \end{align*}
    Since \(m/\gcd(i,m)\) and \(i/\gcd(i,m)\) are relatively prime, it follows that \(k=m/\gcd(i,m)\mid \ell\text{,}\) and hence that \(k\leq \ell\text{,}\) since \(k\) and \(\ell\) are positive. This proves \(k\) is the smallest positive integer such that \((r^i)^k=1\text{,}\) and thus that \(\ord r^i=k\text{.}\)
