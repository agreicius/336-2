Theorem 1.1.15. Units modulo \(n\).

Let \(n\) be a positive integer.
  1. An element \([a]_n\in \Z/n\Z\) is a unit if and only if \(\gcd(a,n)=1\text{.}\) In other words:
    \begin{align} (\Z/n\Z)^*\amp =\{[a]_n\in \Z/n\Z\mid \gcd(a,n)=1\}\text{.}\tag{1.1.8} \end{align}
    As a result,
    \begin{align} \abs{(\Z/n\Z)^*} \amp = \varphi(n) \text{,}\tag{1.1.9} \end{align}
    where \(\varphi\) is the Euler totient function.
    🔗
    🔗
  2. Assume \(a\in \Z\) is relatively prime to \(n\text{.}\) Given integers \(b\) and \(c\) satisfying \(ab+cn=1\text{,}\) as guaranteed by the Euclidean algorithm, we have
    \begin{align} [a]_n^{-1} \amp = [b]_n\text{.}\tag{1.1.10} \end{align}
    🔗
    🔗
  3. If \([a]_n, [b]_n\in (\Z/n\Z)^*\text{,}\) then \([a]_n\cdot[b]_n\in (\Z/n\Z)^*\text{.}\) In other words, \((\Z/n\Z)^*\) is closed under multiplication.
    🔗
    🔗
  4. If \([a]_n\in (\Z/n\Z)^*\text{,}\) then
    \begin{align} [a]_n\cdot [b]_n= [a]_n\cdot [c]_n \amp \text{ if and only if } [b]_n=[c]_n\tag{1.1.11} \end{align}
    for all \([b]_n, [c]_n\in \Z/n\Z\text{.}\) In other words, invertible elements of \(\Z/n\Z\) can be cancelled multiplicatively.
    🔗
    🔗
🔗
🔗

Proof.

  1. Assume \([a]_n\in (\Z/p\Z)^*\text{,}\) and let \(d=\gcd(a,n)\text{.}\) By definition, there is an element \([b]_n\) such that \([a]_n\cdot [b]_n=[ab]_n=[1]_n\text{.}\) By Theorem 1.1.4, this implies \(ab\equiv 1\pmod n\text{,}\) or equivalently, \(ab=1+cn\) for some \(c\in \Z\text{.}\) But then we have \(ba-cn=1\text{.}\) Since \(d=\gcd(a,n)\) divides \(a\) and \(n\text{,}\) we have \(d\mid 1\text{,}\) and thus \(d=1\text{.}\) Thus if \([a]_n\) is a unit, then \(\gcd(a,n)=1\text{.}\)
    🔗
    For the other direction assume \(\gcd(a,n)=1\text{.}\) From the Euclidean algorithm, we know there are integers \(b\) and \(c\) such that \(ba+cn=1\text{.}\) But then \(ab\equiv 1\pmod n\text{,}\) or equivalently, \([a]\cdot [b]_n=[ab]_n=[1]_n\text{.}\) We conclude that \([b]_n=[a]_n^{-1}\text{.}\)
    🔗
    We conclude that \((\Z/n\Z)^*=\{[a]_n\in \Z/n\Z\mid \gcd(a,n)=1\}\text{,}\) as desired. Furthermore, taking our representatives to be least positive residues, we see that
    \begin{align*} (\Z/n\Z)^* \amp = \{[a]_n\in \Z/n\Z\mid \gcd(a,n)=1, 0\leq a\leq n-1\} \text{,} \end{align*}
    from which it follows immediately from the definition of the Euler totient function and Theorem 1.1.4 that
    \begin{align*} \abs{(\Z/n\Z)^*} \amp = \varphi(n) \text{.} \end{align*}
    🔗
    🔗
  2. As we saw in the previous argument, if \(ba+cn=1\text{,}\) then \([ab]_n=[1]_n\text{,}\) and thus \([b]_n=[a]_n^{-1}\text{.}\)
    🔗
    🔗
  3. This follows from (1) and the fact that if \(a\) and \(b\) are both relatively prime to \(n\text{,}\) then so is \(ab\text{.}\)
    🔗
    🔗
  4. The implication
    \begin{align*} [b]_n=[c]_n \amp \implies [a]_n\cdot [b]_n=[a]_n\cdot [c]_n \end{align*}
    is clear. For the other direction, we have
    \begin{align*} [a]_n\cdot [b]_n=[a]_n\cdot [c]_n \amp \implies [a]_n^{-1}\cdot ([a]_n\cdot [b]_n)=[a]^{-1}\cdot ([a]_n\cdot [c]_n)\\ \amp \implies ([a]_n^{-1}\cdot [a]_n)\cdot [b]_n=([a]^{-1}\cdot [a]_n)\cdot [c]_n \\ \amp \implies [1]_n\cdot [b]_n=[1]_n\cdot [c]_n\\ \amp \implies [b]_n=[c]_n\text{.} \end{align*}
    🔗
    🔗
