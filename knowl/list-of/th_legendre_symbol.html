Theorem 1.5.7. Quadratic residues.

Let \(p\) be an odd prime.
  1. If \(a\equiv b\pmod p\text{,}\) then \(\qr{a}{p}=\qr{b}{p}\text{.}\)
    🔗
    🔗
  2. \(\qr{ab}{p}=\qr{a}{p}\qr{b}{p}\) for all \(a,b\in \Z\text{.}\)
    🔗
    🔗
  3. \(\qr{a^2}{p}=1\) for all \(a\in \Z\) with \(p\nmid a\text{.}\)
    🔗
    🔗
  4. Exactly \((p-1)/2\) (i.e., half) of the elements of \(\{1,2,\dots, p-1\}\) are quadratic residues modulo \(p\text{.}\)
    🔗
    🔗
  5. Euler’s criterion.
    \(\qr{a}{p}\equiv a^{(p-1)/2}\pmod p \) for all \(a\in \Z\text{.}\)
    🔗
    🔗
🔗
🔗

Proof.

  1. This is essentially by definition and the observation that \(\qr{a}{p}=0\) if and only if \(p\mid a\text{,}\) if and only if \(a\equiv 0\pmod p\text{.}\)
    🔗
    🔗
  2. It is clear that if either of \(a\) and \(b\) are divisible by \(p\text{,}\) then \(0=\qr{ab}{p}=\qr{a}{p}\qr{b}{p}\text{.}\) Assume now that \(a\) and \(b\) are relatively prime to \(p\text{.}\) Let \(r\) be a primitive root modulo \(p\text{,}\) and let \([a]_p=r^i\) and \([b]_p=r^j\text{.}\) We know that an element \(r^k\in \F_p^*\) is a square if and only if \(k\) is even. It follows that \(\qr{a}{p}=(-1)^i\text{,}\) \(\qr{b}{p}=(-1)^j\text{.}\) Furthermore, since \([ab]_p=r^{i+j}\text{,}\) we have \(\qr{ab}{p}=(-1)^{i+j}\text{.}\) We conclude that
    \begin{align*} \qr{a}{p}\qr{b}{p} \amp = (-1)^i\cdot (-1)^j\\ \amp = (-1)^{i+j}\\ \amp = \qr{ab}{p}\text{.} \end{align*}
    🔗
    🔗
  3. We have \(\qr{a^2}{p}=\qr{a}{p}\qr{a}{p}=\left(\qr{a}{p}\right)^2=1\text{,}\) since \(\qr{a}{p}=\pm 1\text{.}\)
    🔗
    🔗
  4. Let \(s=[a]_p\text{.}\) By Theorem 1.5.2, we know that \(s\) is a square if and only if \(s^{(p-1)/2}=1\text{.}\) Since \(x=s^{(p-1)/2}\) satisfies \(x^2=1\text{,}\) and since the only roots of \(f(x)=x^2-1\) are \([1]_p\) and \([-1]_p\text{,}\) we see that either \(a^{(p-1)/2}\equiv 1\pmod p\) and \(\qr{a}{p}=1\text{,}\) or \(a^{(p-1)/2}\equiv -1\pmod p\) and \(\qr{a}{p}=-1\text{.}\) In both cases we have \(a^{(p-1)/2}\equiv \qr{a}{p} \pmod p\text{,}\) as desired.
    🔗
    🔗
