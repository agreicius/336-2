Theorem 1.15.13. Quadratic ring of integers.

Let \(d\in \Z-\{0,1\}\) be a squarefree integer, and let \(K=\Q(\sqrt{d})\text{.}\)
  1. \(\mathcal{O}_K\) is a subring of \(\C\text{.}\)
    🔗
    🔗
  2. Given \(\alpha\in K\text{,}\) we have \(\alpha\in \mathcal{O}_K\) if and only if \(\alpha+\overline{\alpha}\in \mathcal{O}_K\) and \(\alpha\overline{\alpha}\in \mathcal{O}_K\text{.}\)
    🔗
    🔗
  3. We have
    \begin{align*} \mathcal{O}_K \amp =\begin{cases} \Z[\sqrt{d}]=\{a+b\sqrt{d}\mid a,b\in \Z\} \amp \text{if } d\equiv 2,3\pmod 4\\ \Z[\omega]=\{a+b\omega\mid a,b\in \Z\}, \text{ where } \omega=\tfrac{1}{2}+\tfrac{\sqrt{d}}{2} \amp \text{if } d\equiv 1\pmod 4 \text{.}\end{cases} \end{align*}
    🔗
    🔗
🔗
🔗

Proof.
