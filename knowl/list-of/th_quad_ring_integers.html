Theorem 1.15.13. Quadratic ring of integers.

Let \(d\in \Z-\{0,1\}\) be a squarefree integer, and let \(K=\Q(\sqrt{d})\text{.}\)
  1. \(\mathcal{O}_K\) is a subring of \(\C\text{.}\)
    🔗
    🔗
  2. Given \(\alpha\in K\text{,}\) we have \(\alpha\in \mathcal{O}_K\) if and only if \(\alpha+\overline{\alpha}\in \ZZ\) and \(\alpha\overline{\alpha}\in \Z\text{.}\)
    🔗
    🔗
  3. We have
    \begin{align} \mathcal{O}_K \amp =\begin{cases} \Z[\sqrt{d}]=\{a+b\sqrt{d}\mid a,b\in \Z\} \amp \text{if } d\equiv 2,3\pmod 4\\ \Z[\omega]=\{a+b\omega\mid a,b\in \Z\}, \text{ where } \omega=\tfrac{1}{2}+\tfrac{\sqrt{d}}{2} \amp \text{if } d\equiv 1\pmod 4 \text{.}\end{cases}\tag{1.15.14} \end{align}
    🔗
    🔗
🔗
🔗

Proof.

  1. This follows easily from the fact the set of algebraic integers is closed under addition and multiplication, as is \(K\text{.}\)
    🔗
    🔗
  2. Note that \(\alpha\) is a root of the monic polynomial
    \begin{align*} f(x) \amp =(x-\alpha)(x-\overline{\alpha})=x^2-(\alpha+\overline{\alpha})x+\alpha\overline{\alpha}\text{.} \end{align*}
    If \(\alpha+\overline{\alpha}\in \Z\) and \(\alpha\overline{\alpha}\in \Z\text{,}\) then \(f\in \Z[x]\) and thus \(\alpha\in \mathcal{O}_K\text{.}\)
    🔗
    Conversely, assume \(\alpha\in\mathcal{O}_K\text{,}\) and suppose \(f(\alpha)=0\) for the polynomial \(f(x)=\anmonic\in \Z\text{.}\) Since \(\overline{a_i}=a_i\) for all \(0\leq i\leq n-1\text{,}\) we see that
    \begin{align*} 0 \amp =\overline{f(\alpha)}\\ \amp =\overline{a_n}\overline{\alpha}^n+\overline{a_{n-1}}\overline{\alpha}^{n-1}+\cdots +\overline{a_1}\alpha+\overline{a_0}\\ \amp = a_n\overline{\alpha}^n+a_{n-1}\overline{\alpha}^{n-1}+\cdots +a_1\alpha+a_0\\ \amp = f(\overline{\alpha})\text{.} \end{align*}
    Thus \(\overline{\alpha}\) is also a root of \(f\text{,}\) showing tha that \(\overline{\alpha}\in \mathcal{O}_K\text{.}\) It follows that \(\alpha+\overline{\alpha}\) and \(\alpha\overline{\alpha}\) are elements of \(\mathcal{O}_K\text{.}\) Lastly, since
    \begin{align*} \overline{\alpha+\overline{\alpha}} \amp =\overline{\alpha}+\alpha\\ \overline{\alpha\overline{\alpha}} \amp = \overline{\alpha}\alpha\text{,} \end{align*}
    we see that \(\alpha+\overline{\alpha},\alpha\overline{\alpha}\in \Q\text{.}\) From (1) of Theorem 1.15.11, we conclude that \(\alpha+\overline{\alpha},\alpha\overline{\alpha}\in \Z\text{.}\)
    🔗
    🔗
  3. Using the condition from (2), it is easy to see that elements of the rings on the right side of the equality (1.15.14) are elements of \(\mathcal{O}_K\text{.}\) It remains to show that the reverse inclusion.
    🔗
    Assume \(\alpha=q_1+q_2\sqrt{d}\in \mathcal{O}_K\text{.}\) From (2) it follows that \(\alpha+\overline{\alpha},\alpha\overline{\alpha}\in \Z\text{.}\) We have
    \begin{align*} \alpha+\overline{\alpha} \amp =2q_1\\ \alpha\overline{\alpha} \amp =q_1^2-q_2^2d\text{.} \end{align*}
    Since \(2q_1\in \Z\text{,}\) we have \(q_1=m/2\) for some integer \(m\text{.}\)
    🔗
    Assume first that \(m\) is even, so that \(q_1\in \Z\text{.}\) It follows that \(q_2^2d=a\in \Z\text{.}\) Using the fact that \(d\) is square free, we can show that \(q_2\in \Z\) in this case. Indeed, writing \(q_2=r/s\) with \(\gcd(r,s)=1\text{,}\) we have \(r^2d=s^2a\text{.}\) If \(s\) had a prime divisor \(p\text{,}\) \(p^2\mid s^2a\text{,}\) and since \(\gcd(r,s)=1\text{,}\) we’d have \(p^2\mid d\text{.}\) Thus when \(q_1\in \Z\) is an integer, \(q_2\in \Z\text{,}\) and we have \(\alpha\in \Z[\sqrt{d}]\subseteq \Z[\omega]\text{.}\)
    🔗

    \begin{equation*} \frac{m^2}{4}-q_2^2d\in \Z\text{,} \end{equation*}

    \begin{equation*} \frac{m^2}{4}-\frac{n^2}{4}d\in \Z\text{,} \end{equation*}
