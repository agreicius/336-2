Definition 1.15.14. Units of \(\mathcal{O}_K\).

Let \(d\in \Z-\{0,1\}\) be squarefree, and let \(K=\Q(\sqrt{d})\text{.}\) An element \(\alpha \in \mathcal{O}_K\) is a unit (or invertible) if there is an element \(\beta\in \mathcal{O}_K\) such that \(\alpha\beta=1\text{.}\) The group of units of \(\mathcal{O}_K\) is the set \(\mathcal{O}_K^*\) of all units: i.e.,
\begin{equation*} \mathcal{O}_K^*=\{\alpha\in \mathcal{O}_K\mid \alpha\beta=1 \text{ for some } \beta\in \mathcal{O}_K^*\}\text{.} \end{equation*}
