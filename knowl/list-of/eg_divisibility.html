Example 1.11.11. Divisibility in \(\Z[i]\).

For the given \(\alpha, \beta\in \Z[i]\) determine whether \(\alpha\mid \beta\) and whether \(\beta\mid \alpha\text{.}\)
  1. \(\alpha=1+2i\text{,}\) \(\beta=3+i\)
    🔗
    🔗
  2. \(\alpha=1+2i\text{,}\) \(\beta=2-i\)
    🔗
    🔗
🔗
Solution.
One way of answering this type of question is to use some logic similar to Remark 1.11.6, zooming out a bit to the larger number system \(\C\text{.}\) In general, for nonzero elements \(\alpha, \beta\in \C\text{,}\) we have \(\beta=\gamma \alpha\) for if and only if \(\gamma=\alpha^{-1}\beta\text{.}\) Thus we have \(\alpha\mid \beta\) in a subring \(R\) if and only if \(\alpha^{-1}\beta\in R\text{.}\)
🔗
Working in \(\Z[i]\text{,}\) we can also use the norm function to give a necessary (though not necessarily sufficient condition) for divisibility: namely, if \(\alpha\mid \beta\text{,}\) then we have \(\beta=\alpha\gamma\) for some \(\gamma\in \Z[i]\text{,}\) in which case \(N(\beta)=N(\alpha)N(\gamma)\) and \(N(\alpha)\mid N(\beta)\text{.}\)
🔗
We illustrate both techniques in the two examples.
  1. Since \(N(\alpha)=5\) and \(N(\beta)=10\text{,}\) we see that \(\beta\nmid \alpha\text{,}\) but possibly \(\alpha\mid \beta\text{.}\) To decide the latter question, we compute
    \begin{equation*} \alpha^{-1}\beta=(1/5-(2/5)i)(3+i)=1-i\in \Z[i]\text{.} \end{equation*}
    Thus \(\alpha\mid \beta\text{.}\)
    🔗
    🔗
  2. In this case we have \(N(\alpha)=N(\beta)=5\text{.}\) Some arithmetic similar to the above shows that \(\alpha=-i\beta\) and \(\beta=-i\alpha\text{.}\) We conclude that \(\alpha\mid \beta\) and \(\beta\mid \alpha\text{.}\)
