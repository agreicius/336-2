Definition 1.5.1. \(n\)-th power.

Let \(n\) be a positive integer, and let \(p\) be a prime. An element \(r\in \F_p\) is an \(n\)-th power if there is an \(s\in \F_p\) satisfying \(s^n=r\text{.}\) We call \(s\) an \(n\)-th root of \(r\) in this case.
🔗
Similarly, we say an integer \(a\in \Z\) is an \(n\)-th power modulo \(p\) if \([a]_p\) is an \(n\)-th power in \(\F_p\text{:}\) equivalently, if there is an integer \(b\in \Z\) such that \(b^n\equiv a \pmod p\text{.}\) We call \(b\) an \(n\)-th root of \(a\) modulo \(p\) in this case.
