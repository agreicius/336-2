Proposition 1.15.15. Units of \(\mathcal{O}_K\).

Let \(d\in \Z-\{0,1\}\) be squarefree, and let \(K=\Q(\sqrt{d})\text{.}\) An element \(\alpha\in \mathcal{O}_K\) if and only if \(N_K(\alpha)=\pm 1\text{.}\) In other words
\begin{equation*} \mathcal{O}_K=\{\alpha\in \mathcal{O}_K\mid N_K(\alpha)=\pm 1\}\text{.} \end{equation*}
🔗
🔗

Proof.

We prove both implications separately.
🔗
If \(\alpha\in \mathcal{O}_K^*\text{,}\) then there is an element \(\beta\in \mathcal{O}_K\) such that \(\alpha\beta=1\text{.}\) Taking norms, we conclude that \(N_K(\alpha)N_K(\beta)=1\text{.}\) Since \(N_K(\alpha)\) and \(N_K(\beta)\) are both integers by Theorem 1.15.13, and since the only divisors of \(1\) are \(\pm 1\text{,}\) we conclude that \(N_K(\alpha)\in \{-1,1\}\text{.}\)
🔗
Conversely, if \(N_K(\alpha)=\alpha\overline{\alpha}=\pm 1\text{,}\) then we see that \(\alpha(\pm \overline{\alpha}}=1\) and thus \(\pm \overline{\alpha}\) is an inverse of \(\alpha\) in \(\C\text{.}\) Since \(\alpha\in \mathcal{O}_K\) if and only if \(\overline{\alpha}\in \mathcal{O}_K\text{,}\) we see that \(\pm \overline{\alpha}\in \mathcal{O}_K\text{,}\) showing that \(\alpha\in \mathcal{O}_K^*\text{.}\)
