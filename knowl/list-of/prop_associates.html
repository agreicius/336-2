Proposition 1.11.12.

Let \(\alpha\) and \(\beta\) be nonzero elements of \(R\text{.}\) The following statements are equivalent.
  1. \(\alpha\) is associate to \(\beta\text{.}\)
    🔗
    🔗
  2. \(\beta\) is associate to \(\alpha\text{.}\)
    🔗
    🔗
  3. \(\alpha\mid \beta\) and \(\beta\mid \alpha\) in \(R\text{.}\)
    🔗
    🔗
🔗
🔗

Proof.

Implication: (i) implies (ii). If \(\alpha\) is associate to \(\beta\text{,}\) then there is a unit \(u\in R^*\) such that \(\alpha=u \beta\text{.}\) Since \(u\in R^*\text{,}\) we have \(u^{-1}\in R^*\) and \(\beta=u^{-1}\alpha\text{.}\) Thus \(\beta\) is associate to \(\alpha\text{.}\)
🔗
Implication: (ii) implies (i). If \(\beta\) is associate to \(\alpha\text{,}\) we have \(\beta=u \alpha\) with \(u\in R^*\text{.}\) In particular, this means \(\alpha\mid \beta\text{.}\) Since \(u\in R^*\text{,}\) there is an element \(u^{-1}\in R\) such that \(uu^{-1}=1\text{.}\) It follows that \(\alpha=u^{-1}\beta\) and \(\beta\mid \alpha\text{.}\)
🔗
Implication: (iii) implies (i). If \(\alpha \mid \beta\) and \(\beta\mid \alpha\text{,}\) we have
\begin{align*} \beta \amp = \gamma \alpha\\ \alpha \amp = \delta \beta \end{align*}
for some elements \(\gamma, \delta\in R\text{.}\) But then we have
\begin{equation*} \alpha=\delta \beta=(\delta\gamma) \alpha\text{.} \end{equation*}
Since \(\alpha\) is nonzero, it has an inverse \(\alha^{-1}\in \C\) (not necessarily an element of \(R\)), that allows us to multiplicatively cancel \(\alpha\) to obtain \(1=\delta\gamma\text{.}\) It follows that \(\delta\in R^*\) and \(\gamma\in R^*\text{,}\) and hence that \(\alpha\) is a associate to \(\beta\text{.}\)
