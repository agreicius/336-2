Proposition 1.12.7. Units of \(\Z[i]\).

We have
\begin{align} (Z[i])^* \amp =\{\alpha\in \Z[i]\mid N(\alpha)=1\}=\{1,-1,i,-i\}\text{.}\tag{1.12.3} \end{align}
🔗
🔗

Proof.

To prove the first set equality, we must show that \(\alpha\) is a unit if and only if \(N(\alpha)=1\text{.}\)
🔗
First assume \(\alpha\in \Z[i]\text{.}\) By definition, there is an element \(\beta\in \Z[i]\) such that \(\alpha\beta=1\text{.}\) Applying \(N\) to both sides of this equation, we see that \(N(\alpha)N(\beta)=N(1)=1\text{.}\) Since \(N(\alpha)\) and \(N(\beta)\) are integers, we conclude that \(N(\alpha)\mid 1\) in \(\Z\text{,}\) and hence that \(N(\alpha)=\pm 1\text{.}\) Since \(N(\alpha)\geq 0\text{,}\) we conclude that \(N(\alpha)=1\text{.}\)
🔗
Now assume \(N(\alpha)=1\text{.}\) By definition this means \(\alpha\overline{\alpha}=1\text{.}\) Since \(\overline{\alpha}\in \Z[i]\text{,}\) we conclude that \(\alpha\in (\Z[i])^*\text{.}\)
🔗
We now know that \(\alpha=a+bi\in (\Z[i])^*\) if and only if \(N(\alpha)=a^2+b^2=1\text{.}\) Since \(a\) and \(b\) are integers, it is easy to see that we must have \((a,b)\in \{(\pm 1,0),(0,\pm 1)\}\text{,}\) and hence that \((\Z[i])^*=\{1,-1,i,-i\}\text{,}\) as claimed.
