Example 1.2.5. Primitive roots.

Compute the order of all elements of \((\Z/n\Z)^*\) for the given modulus \(n\text{,}\) and decide whether \((\Z/n\Z)^*\) has a primitive root.
🔗
Additionally, if a primitive root \(r\in (\Z/n\Z)^*\) does exist, verify the order formula for each power \(r^i\) given in Proposition 1.2.3.
  1. \(\displaystyle n=7\)
    🔗
    🔗
  2. \(\displaystyle n=12\)
    🔗
    🔗
  3. \(\displaystyle n=25\)
    🔗
    🔗
🔗
Solution.
For each modulus below, we will drop most of our notational baggage and simply write \(a\) instead of \([a]_n\text{.}\)
  1. We have \((\Z/7\Z)^*=\{1,2,3,4,5,6\}\text{.}\) Computing successive powers of these elements leads to the following order computations:
    \begin{align*} \ord 1 \amp =1 \amp \ord 2=3\amp \ord 3\amp =6\\ \ord 4 \amp = 3 \amp \ord 5\amp =6 \amp \ord 6\amp =2\text{.} \end{align*}
    Since \(\ord 3=\ord 5=6\text{,}\) we see that both of these elements are primitive roots in \(\Z/7\Z\text{.}\) Let’s verify that \(\angvec{3}=\angvec{5}=(\Z/7\Z)^*\text{:}\)
    \begin{align*} \angvec{3} \amp =\{1, 3, 3^2, 3^3, 3^4, 3^5, 3^6\}\\ \amp = \{1,3,2,6, 4,5\}=(\Z/7\Z)^*\\ \angvec{5} \amp = \{1, 5, 5^2, 5^3, 5^4, 5^5, 5^6\} \\ \amp = \{1,5,4,6,2,3\}=(\Z/7\Z)^*\text{.} \end{align*}
    Lastly, let’s verify the order formula for the powers of \(3\text{.}\)
    🔗
    🔗
  2. We have \((\Z/12\Z)^*=\{1,5,7,11\}\) and we easily compute
    \begin{align*} \ord 1 \amp = 1 \amp \ord 5\amp=2\\ \ord 7 \amp =2 \amp \ord 11\amp=2\text{.} \end{align*}
    Since no element has order 4, there is no primitive root for \((\Z/12\Z)^*\text{.}\)
    🔗
    🔗
  3. Since \(\gcd(a,25)=1\) if and only if \(5\nmid a\text{,}\) the units of \(\Z/25\Z\) are the elements of the form \([a]_{25}\text{,}\) where \(5\nmid a\text{.}\) There are \(20=25-5\) of these. Let’s use Sage to see whether any of these have order 20.
    🔗
    We see that \(2\) has order \(20=\abs{(Z\25\Z)^*)}\text{,}\) and thus that \(2\) is a primitive root. We leave it to the reader to verify the order formula for powers of \(2\) as we did in the first part.
    🔗
    On a separate note, notice that there are 8 distinct primitive roots here: \(2,3,8,12,13,17,22,23\text{.}\) In the next sections we will derive a formula for the number of distinct primitive roots modulo, if one does indeed exist.
