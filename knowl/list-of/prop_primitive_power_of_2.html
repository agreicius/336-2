Proposition 1.4.4. Primitive roots for \(2^k\text{,}\) \(k\geq 3\).

Let \(k\geq 3\text{.}\)
  1. \(\ord r\leq 2^{k-2}\) for all \(r\in (\Z/2^k)*\text{.}\) As a result, there is no primitive root for \((\Z/2^k\Z)^*\text{.}\)
    🔗
    🔗
  2. The element \(r=[5]_{2^k}\in (\Z/2^k)^*\) has order \(2^{k-2}\text{.}\) Every element in \((\Z/2^k\Z)^*\) can we written uniquely in the form \(\pm r^{i}\) for some \(0\leq i\leq 2^{k-2}-1\text{.}\)
