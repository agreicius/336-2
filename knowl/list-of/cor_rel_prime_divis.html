Corollary 1.13.8. Relatively prime factors.

Assume \(\alpha,\beta\) are relatively prime elements of \(\Z[i]\text{.}\)
  1. If \(\alpha\mid \beta\gamma\text{,}\) then \(\alpha\mid \gamma\text{.}\)
    🔗
    🔗
  2. If \(\alpha\mid \gamma\) and \(\beta\mid \gamma\text{,}\) then \(\alpha\beta\mid \gamma\text{.}\)
    🔗
    🔗
🔗
🔗

Proof.

Since \(\alpha\) and \(\beta\) are relatively prime, we can write \(1=\delta \alpha+\omega\beta\) some \(\delta, \omega\in R\text{.}\) We will make use of this expression in both parts below.
  1. Assume that \(\alpha\mid \beta\gamma\) for some \(\gamma in R\text{.}\) By definition this means \(\beta\gamma=\epsilon \alpha\) for some \(\epsilon\in R\text{.}\) It follows that
    \begin{align} \gamma \amp =\gamma (\delta\alpha+\omega\beta)\tag{1.13.2}\\ \amp = (\gamma\delta)\alpha+\omega \ beta\gamma\tag{1.13.3}\\ \amp = (\gamma\delta+\omega\epsilon)\alpha\text{,}\tag{1.13.4} \end{align}
    and thus that \(\alpha\mid \gamma\text{.}\)
    🔗
    🔗
  2. Assume that \(\alpha\mid \gamma\) and \(\beta\mid \gamma\) for some \(\gamma in R\text{.}\) By definition, this means \(\gamma=\epsilon \alpha\) and \(\gamma=\phi \beta\) for some \(\epsilon, \phi\in R\text{.}\) From \(1=\delta\alpha+\omega\beta\) it now follows that
    \begin{align*} \gamma \amp =\gamma\delta\alpha+\gamma\omega\beta\\ \amp = (\delta\phi)\alpha\beta+(\gamma\epsilon)\alpha\beta\text{,} \end{align*}
    and we see that \(\alpha\beta\mid \gamma\text{.}\)
