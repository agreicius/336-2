Lemma 1.16.6. Units in \(\mathcal{O}_K^*\cap (1,\infty)\).

Let \(d\in \Z-\{0,1\}\) be a positive squarefree integer, and let \(K=\Q(\sqrt{d})\text{.}\) Assume \(u=q_1+q_2\sqrt{d}\in \mathcal{O}_K^*-\{1,-1\}\text{.}\)
  1. We have
    \begin{align} u \amp \in (-\infty, -1)\cup (-1,0)\cup (0,1)\cup (1,\infty) \text{.}\tag{1.16.2} \end{align}
    🔗
    🔗
  2. We have
    \begin{align} \{u,-u,u^{-1},-u^{-1}\} \amp = \{q_1+q_2\sqrt{d}, q_1-q_2\sqrt{d},-q_1+q_2\sqrt{d}, -q_1-q_2\sqrt{d}\}\subseteq \mathcal{O}_K^* \text{.}\tag{1.16.3} \end{align}
    🔗
    🔗
  3. Each interval appearing in (1.16.2) contains exactly one of the elements \(\{u,-u,u^{-1},-u^{-1}\}\text{.}\)
    🔗
    🔗
  4. \(u\in (1,\infty)\) if and only if \(q_1\) and \(q_2\) are positive.
    🔗
    🔗
🔗
🔗

Proof.

  1. Since \(d> 0\text{,}\) \(K\subseteq \R\text{.}\) Since \(u\in \mathcal{O}_K^*\text{,}\) we must have \(u\ne 0\text{.}\) And by assumption we have \(u\ne \pm 1\text{.}\) It follows that \(\alpha\in \R-\{-1,1,0\}\)
    🔗
    🔗
  2. Clearly \(u\in \mathcal{O}_K^*\) if and only if \(-u\in \mathcal{O}_K^*\text{.}\) Furthermore, since \(u\in \mathcal{O}_K\) if and only if \(\overline{u}\in \mathcal{O}_K\text{,}\) and since \(u^{-1}=\pm \overline{u}\text{,}\) we have \(u\in \mathcal{O}_K\) if and only if \(u^{-1}\in \mathcal{O}_K\text{.}\) Lastly, the set equality follows easily from our formula for inverses and the fact that \(N_K(u)=\pm 1\text{.}\)
    🔗
    🔗
  3. In general, for any \(\alpha\in \R-\{-1,0,1\}\text{,}\) we have
    \begin{align*} \alpha\in (1,\infty) \amp\iff \alpha^{-1}\in (0,1) \\ \amp \iff -\alpha^{-1}\in (-1, 0)\\ \amp \iff -\alpha\in (-\infty,-1) \text{,} \end{align*}
    and thus each of the four intervals of (1.16.2) contains exactly one of the elements of \(\{u, u{-1},-u^{-1},-u\}\text{.}\)
    🔗
    🔗
  4. First note that since \(N_K(u)=q_1^2-q_2^2d=\pm 1\) and \(d\geq 2\text{,}\) we cannot have \(q_1=0\text{;}\) and since \(u\ne \pm 1\text{,}\) we cannot have \(q_2=0\text{.}\)
    🔗
    Next, since \(u'=\abs{q_1}+\abs{q_2}\sqrt{d}\) is clearly the maximum element of \(\{u,-u,u^{-1},-u^{-1}\}\text{,}\) and since exactly one of these elements lies in \((1,\infty)\text{,}\) \(u'\) must be the sole element of \((1,\infty)\text{.}\) It follows that \(u\in (1,\infty)\) if and only if \(u=u'\) if and only if \(q_1\) and \(q_2\) are positive.
    🔗
    🔗
