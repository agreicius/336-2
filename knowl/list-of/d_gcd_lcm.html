Definition 1.13.3. Greatest common divisor and least common multiple.

Let \(R\) be a subring of \(\C\text{.}\) Given elements \(\alpha, \beta\in R\) a greatest common divisor (or GCD) of \(\alpha\) and \(\beta\) is an element \(\delta\in R\) satisfying the following conditions:
  1. \(\delta\mid \alpha\) and \(\delta\mid \beta\text{;}\)
    🔗
    🔗
  2. if \(\delta'\in R\) satisfies \(\delta'\mid \alpha\) and \(\delta'\mid \beta\text{,}\) then \(\delta'\mid \delta\text{.}\)
    🔗
    🔗
🔗
A least common multiple (or LCM) of \(\alpha\) and \(\beta\) is an element \(\mu\in R\) satisfying the following conditions:
  1. \(\alpha\mid \mu\) and \(\beta\mid \mu\text{;}\)
    🔗
    🔗
  2. if \(\mu'\in R\) satisfies \(\alpha\mid \mu'\) and \(\beta\mid \mu'\text{,}\) then \(\mu\mid \mu'\text{.}\)
    🔗
    🔗
More generally, a greatest common divisor of elements \(\alpha_1,\alpha_2,\dots, \alpha_n\) of \(R\) is an element \(\delta\in R\) such that \(\delta\mid \alpha_i\) for all \(i\text{,}\) and if \(\delta'\in R\) satisfies \(\delta'\mid \alpha_i\) for all \(i\text{,}\) then \(\delta'\mid \delta\text{.}\) Lastly, we say that elements \(\alpha_1,\alpha_2,\dots, \alpha_n\) are relatively prime if they have \(1\) as a GCD.
