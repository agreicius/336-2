Theorem 1.17.3. Pell’s equation.

Let \(d\) be a positive integer that is not a square. There is a solution to the equation
\begin{equation*} x^2-dy^2=1 \end{equation*}
in positive integers.
🔗
🔗

Proof.

Let \(X\) be the set of all pairs \((m,n)\) satisfying
\begin{equation*} \abs{m-n\sqrt{d}}=\abs{n\sqrt{d}-m}< \frac{1}{n} \end{equation*}
The set \(X\) is infinite by Theorem 1.17.2. The proof will proceed in a number of steps.
  1. We show that there is a nonzero integer \(M\) such that \(m^2-n^2d=M\) for infinitely many pairs \((m,n)\in X\text{.}\)
    🔗
    🔗
  2. We show that there is an infinite subset \(Y\) of \(X\) satisfying the following properties:
    • \(m^2-n^2d=1\) for all \((m,n)\in Y\)
      🔗
      🔗
    • If \((m,n), (m',n')\in Y\text{,}\) then \(m\equiv m'\pmod \abs{M}\) and \(n\equiv n'\pmod \abs{M}\text{.}\)
      🔗
      🔗
    🔗
    🔗
  3. Associating to pairs \((m,n)\in Y\) the element \(m+n\sqrt{d}\text{,}\) we will do some algebra in \(\Q(\sqrt{d})\) to construct an element \(u=a+b\sqrt{d}\ne \pm 1\) satisfying \(a^2-b^2d=1\text{.}\)
    🔗
    🔗
🔗
Step 1.
For all \((m,n)\in X\text{,}\) we have
\begin{align*} \abs{m^2-n^2d} \amp =\abs{m+n\sqrt{d}}\abs{m-n\sqrt{d}}\\ \amp =\abs{(m-n\sqrt{d})+2n\sqrt{d}}\abs{m-n\sqrt{d}}\\ \amp < (\abs{m-n\sqrt{d}}+\abs{2n\sqrt{d}})\cdot \frac{1}{n}\\ \amp < (\frac{1}{n}+2\sqrt{d})\cdot \frac{1}{n} \\ \amp < 1+2\sqrt{d}\text{.} \end{align*}
Since \(m^2-n^d\in \Z\) for all \((m,n)\in X\text{,}\) it follows that \(m^2-n^2d\) assumes finitely many values as \((m,n)\) ranges over \(X\text{.}\) Since \(X\) is infinite, there is one value \(M\in \Z\) such that \(m^2-n^2d=M\) for infinitely-many elements of \(X\text{.}\) Since \(\sqrt{d}\) is irrational, it follows that \(M\ne 0\text{.}\)
🔗
🔗
Step 2.
Let \(X'\) be the set of all \((m,n)\) satisfying \(m^2-n^2d=M\text{,}\) and consider the map
\begin{align*} X'\amp \rightarrow \{0,1,\dots, \abs{M}-1\}\times \{0,1,\dots, \abs{M}-1\} \amp \\ (m,n) \amp \mapsto (m\boldmod \abs{M}, n\boldmod \abs{M}) \amp \end{align*}
Since \(X'\) is infinite and the set of pairs \((a,b)\) with \(a,b\in \{0,1,\dots, \abs{M}-1\}\) is finite, there must be infinitely many pairs \((m,n)\in X'\) that get set sent to the same pair \((a,b)\in \{0,1,\dots, \abs{M}-1\}\times \{0,1,\dots, \abs{M}-1\}\text{.}\) Let \(Y\) be the set of all such pairs \((m,n)\in X'\text{.}\) Any pair \((m,n)\) in \(Y\) satisfies \(m^2-n^2d=M\text{,}\) and any two pairs in \((m,n),(m',n')\in Y\) satisfy
\begin{align*} m \amp \equiv m'\equiv a\\ n \amp\equiv n'\equiv b \text{.} \end{align*}
🔗
🔗
Step 3.
Since \(Y\) is infinite, we can find pairs \((m,n)\) and \((m',n')\) satisfying \(m^2-n^2d=(m')^2-(n')^2d=M\) and \(m\ne \pm m'\text{,}\) \(n\ne \pm n'\text{.}\) Let \(K=\Q(\sqrt{d})\) and consider the elements \(\alpha=m+n\sqrt{d}\text{,}\) \(\beta=m'-n'\sqrt{d}\) of \(\mathcal{O}_K\text{.}\) We have
\begin{equation*} \alpha\beta=(mm'-nn'd)+(m'n-mn')\sqrt{d}\text{.} \end{equation*}
Since
\begin{align*} mm'-nn'd \amp \equiv m^2-n^2d= M \equiv 0\pmod \abs{M}\\ m'n-mn' \amp \equiv mn-mn \equiv 0 \pmod \abs{M}\text{,} \end{align*}
we see that \(M\mid \alpha\beta\text{,}\) and thus we have
\begin{equation*} \alpha\beta=Mu \end{equation*}
for some \(u\in \mathcal{O}_K\text{.}\) Now take norms of both sides and simplify:
\begin{align*} N_K(\alpha)N_K(\beta) \amp = N_K(M)N_K(u)\\ M\cdot M \amp = M^2N_K(u)\\ 1 \amp =N_K(u) \amp (M\ne 0)\text{.} \end{align*}
We conclude that \(u=a+b\sqrt{d}\) satisfies \(a^2-b^2d=1\text{,}\) and is thus a solution to Pell’s equation. It remains only to show that \(u\ne \pm 1\text{.}\) We have
\begin{align*} u=\pm 1 \amp \implies \alpha\beta=\pm M \\ \amp \implies \alpha\beta\overline{\beta}=\pm M\overline{\beta}\\ \amp \implies \alpha\cdot M=\pm M \overline{\beta}\\ \amp \implies \alpha=\pm \overline{\beta}\text{.} \end{align*}
Since the last equality is excluded by our condition \(m\ne \pm m'\text{,}\) \(n\ne\pm n'\text{,}\) we see that \(u\ne \pm 1\text{.}\)
