Corollary 1.16.7. Unit group of real quadratic extensions.

Let \(d\in \Z-\{0,1\}\) be a positive squarefree integer, let \(K=\Q(\sqrt{d})\text{,}\) and let \(u=q_1+q_2\sqrt{d}\) and \(u'=q_1'+q_2'\sqrt{d}\) be elements of \(\mathcal{O}_K^*\cap (1,\infty)\text{.}\) We have \(u< u'\) if and only if \(q_1< q_1'\text{.}\)
🔗
As a result, the fundamental unit \(\epsilon_d\) described in Theorem 1.16.4 is the element \(q_1+q_2\sqrt{d}\) of \(\mathcal{O}_K^*\cap (1,\infty)\) with minimal first coefficient \(q_1\text{.}\)
🔗
🔗

Proof.

Let \(\epsilon\) be the smallest element of \(\mathcal{O}_K\cap(1,\infty)\text{,}\) and write \(\epsilon=r+s\sqrt{d}\text{,}\) where \(r=m/2\) and \(s=n/2\) for some positive integers \(m,n\in \Z\text{.}\) Since \(\mathcal{O}_K^*=\{\pm \epsilon^n\mid n\in \Z\}\text{,}\) and since \(\abs{\epsilon^{n}}\leq 1\) for all \(n\leq 0 \text{,}\) it follows that
\begin{equation*} \mathcal{O}_K^*\cap (1,\infty)=\{\epsilon^n\mid n\geq 1\}\text{.} \end{equation*}
Since furthermore the function \(f(x)=\epsilon^x\) is increasing, we have
\begin{equation*} \epsilon< \epsilon^2< \epsilon^3< \dots\text{.} \end{equation*}
Thus for all \(u,v\in \mathcal{O}_K^*\cap (1,\infty)\text{,}\) we have \(u=\epsilon^m\) and \(v=\epsilon^n\) for some positive integers \(m\) and \(n\text{,}\) and \(u< v\) if and only if \(m< n\text{.}\) Let \(\epsilon^n=r_n+s_n\sqrt{d}\text{.}\) To finish the proof, it suffices to show that \(r_n< r_{n+1}\) for all \(n\geq 1\text{,}\) or equivalently, that \(r_n\) is an increasing function of \(n\text{.}\) Since
\begin{align*} \epsilon^{n+1} \amp =\epsilon \epsilon^{n}\\ \amp =(r_1+s_1\sqrt{d})(r_n+s_n\sqrt{d})\\ \amp =(r_1r_n+s_1s_nd)+(r_1s_n+r_ns_1)\sqrt{d}\text{,} \end{align*}
we see that
\begin{align*} r_{n+1} \amp = r_1r_n+s_1s_nd \amp s_{n+1}\amp=r_1s_n+r_ns_1\text{.} \end{align*}
Recall that since \(\epsilon\in (1,\infty)\text{,}\) we have \(r_1,s_1> 0\text{.}\) If \(r_1> 1\text{,}\) then \(r_{n+1}> r_n+s_1s_nd> r_n\text{.}\) The only other option is \(r_1=\frac{1}{2}\text{.}\) In this case we must have \(d\equiv 1\pmod 4\text{.}\) Furthermore, since
\begin{align*} s_1^2d=r_1^2\pm 1=\frac{1}{4}\pm 1 \amp \text{,} \end{align*}
we must have \(s_1^2d=1/4+1=5/4\text{,}\) from whence it follows that \(d\leq 5\text{.}\) But then \(d=5\) (since \(d\equiv 1\pmod 4\)), and we have \(s_1=1/2\text{.}\) Now we are in the very special case where \(\epsilon=\frac{1}{2}+\frac{\sqrt{5}}{2}\text{.}\) Since \(\epsilon\) satisfies \(f(x)=(x-\epsilon)(x-\overline{\epsilon})=x^2-x-1\text{,}\) it follows that \(\epsilon^2=\epsilon+1\text{,}\) and by induction \(\epsilon^{n+1}=\epsilon^{n}+\epsilon^{n-1}\) for all \(n\geq 2\text{.}\) As a result, looking only at the first coefficients of these powers of \(\epsilon\text{,}\) we have \(r_{n+1}=r_{n}+r_{n-1}\) for all \(n\text{.}\) (And looking at the second coefficients, we see that \(s_{n+1}=s_n+s_{n-1}\text{.}\)) Since \(r_k> 0\) for all \(k\geq 1\text{,}\) we see that \(r_{n+1}> r_n\text{,}\) as desired.
