Corollary 1.13.7. Bezout’s theorem for \(\Z[i]\).

For any two elements \(\alpha, \beta\in \Z[i]\text{,}\) one of which is nonzero, there is a GCD \(\delta\in \Z[i]\text{.}\) Furthermore, we can write
\begin{align} \delta \amp =\gamma \alpha+\omega \beta\tag{1.13.1} \end{align}
for elements \(\gamma, \omega\in \Z[i]\text{.}\)
