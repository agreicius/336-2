Proposition 1.13.13. Primes, norms, and conjugates.

Let \(\pi\in \Z[i]\) be a nonzero non-unit.
  1. \(\pi\) is prime if and only if \(\overline{\pi}\) is prime if and only if \(u\pi\) is prime for any unit \(u\in \Z[i]^*\text{.}\)
    🔗
    🔗
  2. If \(N(\pi)=p\) for \(p\) a prime integer, then \(\pi\) is prime in \(\Z[i]\text{.}\)
    🔗
    🔗
  3. If \(\pi\) is prime in \(\Z[i]\text{,}\) then \(N(\pi)\in \{p,p^2\}\text{,}\) where \(p\) is a prime integer.
    🔗
    🔗
🔗
🔗

Proof.

  1. Proof by delegation. (In other words, this is left to you!) The first equivalence follows easily from the fact that \(\overline{\alpha\beta}=\overline{\alpha}\overline{\beta}\text{.}\) The second follows from the fact that units \(u\) can be cancelled multiplicatively in equations involving Gaussian integers.
    🔗
    🔗
  2. Assume \(N(\pi)=p\) where \(p\) is a prime integer. We will show \(\pi\) is irreducible, hence prime. Given \(\alpha,\beta\in \Z[i]\text{,}\) we have
    \begin{align*} \pi=\alpha\beta \amp \implies p=N(\pi)=N(\alpha)N(\beta)\\ \amp \implies \{N(\alpha),N(\beta)\}=\{1,p\} \amp (p \text{ prime})\\ \amp \implies N(\alpha)=1 \text{ or } N(\beta)=1\\ \amp \implies \alpha\in \Z[i]^* \text{ or } \beta\in \Z[i]^*\text{.} \end{align*}
    Thus \(\pi\) is irreducible/prime, as claimed.
    🔗
    🔗
  3. Assume \(\pi\) is prime/irreducible. Write \(a=N(\pi)\text{,}\) where \(a\) is an integer and \(a> 1\) (since \(\pi\) is not unit). By prime factorization, we can write \(a=\prod_{i=1}^r p_i\) for primes \(p_i\text{.}\) Since \(\pi\mid a\) in \(\Z[i]\text{,}\) it divides \(p_i\) for some \(i\text{.}\) Call \(p=p_i\text{.}\) Since \(\pi\mid p\text{,}\) we have \(p=\pi\alpha\) for some \(\alpha\in \Z[i]\text{.}\) Taking conjugates, we have
    \begin{align*} p\amp=\overline{p}=\overline{\pi}\overline{\alpha} \amp \text{.} \end{align*}
    But then
    \begin{align*} p^2 \amp =pp\\ \amp =\pi\overline{\pi}\alpha\overline{\alpha}\\ \amp = N(\pi)N(\alpha)\text{.} \end{align*}
    Since \(p\) is prime the only factors of \(p^2\) are \(1,p,p^2\text{.}\) Since \(N(\pi)\ne 1\text{,}\) we conclude that \(N(\pi)\in\{p,p^2\}\text{,}\) as claimed.
