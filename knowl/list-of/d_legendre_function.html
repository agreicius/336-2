Definition 1.5.6. Legendre symbol.

Let \(p\) be an odd prime. Given an integer \(a\in \Z\text{,}\) we define the Legendre symbol \(\qr{a}{p}\) as
\begin{equation*} \qr{a}{p}=\begin{cases} 0 \amp \text{if } p\mid a \\ 1 \amp \text{if } p\nmid a \text{ and } a \text{ is a square modulo } p\\ -1 \amp \text{if } p\nmid a \text{ and } a \text{ is not a square modulo } p \text{.}\end{cases} \end{equation*}
The Legendre symbol (with respect to \(p\)) gives rise to a function \(f\colon \Z\rightarrow \{\pm 1\}\) defined as \(f(a)=\qr{a}{p}\text{.}\)
