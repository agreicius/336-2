Theorem 1.13.5. Euclidean algorithm in \(\Z[i]\).

Given elements \(\alpha,\beta\in \Z[i]\text{,}\) one of which is nonzero, we can produce a GCD following the Euclidean algorithm below.
🔗
If \(\alpha\mid \beta\text{,}\) then \(\alpha\) is a GCD of \(\alpha\) and \(\beta\text{.}\) Otherwise, we can produce a sequence of division algorithm instances
\begin{align*} \beta\amp =q_1\alpha+r_1 \amp r_1\amp \ne 0, N(r_1)< N(\alpha) \\ \alpha\amp =q_2r_1+r_2 \amp r_2\amp \ne 0, N(r_2)< N(r_1)\\ \amp \vdots\\ r_{n-1} \amp =q_{n+1}r_{n}+r_{n+1} \amp r_{n+1}\amp\ne 0, N(r_{n+1})< N(r_{n})\\ r_{n} \amp = q_{n+2}r_{n+1}+\boxed{0}\text{,} \end{align*}
and it follows that \(r_{n+1}\) is a GCD of \(\alpha\) and \(\beta\text{.}\) Moreover, beginning with the penultimate equation \(r_{n-1}=q_{n+1}r_n+r_{n+1}\text{,}\) and working our way upward, solving successively for \(r_{k}\) in terms of \(r_{k-1}\) and \(r_{k-2}\text{,}\) we can find elements \(\gamma, \delta\in R\) satisfying
\begin{align*} r_{n+1} \amp =\gamma\alpha+\delta\beta\text{.} \end{align*}
🔗
🔗

Proof.

First note that if \(\beta=q\alpha+r\) for some \(q,r\in \Z[i]\text{,}\) then \(\delta\) is a GCD of \(\alpha\) and \(\beta\) if and only if it is a GCD of \(\alpha\) and \(r\text{,}\) using the usual divisibility arguments.

\begin{align*} r_{k} \amp =q_{k+2}r_{k+1}+r_{k+2}\text{,} \end{align*}
if \(r_{k+2}\ne 0\text{,}\) then we can apply the division algorithm to \(r_{k+2}\) and \(r_{k+1}\text{.}\) Since at each step where the remainder is nonzero, the successive remainder has strictly smaller \(N\)-value, the process must terminate: i.e., at some point we must have \(r_{k}=0\text{.}\)
