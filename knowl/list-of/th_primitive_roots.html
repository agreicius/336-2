Theorem 1.3.7. Primitive roots modulo \(p\).

Let \(p\) be a prime integer. For each positive integer \(k\text{,}\) let \(F_k\) be the set of elements of \((\Z/p\Z)^*\) of order \(k\text{.}\)
  1. If \(k\nmid \phi(p)=p-1\text{,}\) then \(F_k=\emptyset\text{.}\)
    🔗
    🔗
  2. If \(k\mid \phi(p)\text{,}\) then \(\abs{F_k}=\phi(k)\text{.}\)
    🔗
    🔗
  3. In particular, \(\abs{F_{\phi(p)}}=\phi(p-1)\ne 0\text{,}\) and thus there is a primitive root \(r\) of \(\Z/p\Z\text{:}\) i.e., there is an element \(r\in (\Z/p\Z)^*\) satisfying \(\angvec{r}=(\Z/p\Z)^*\text{.}\)
    🔗
    🔗
🔗
🔗

Proof.

  1. We know from Proposition 1.2.3 that \(\ord r\mid \phi(p)\) for all \(p\in \F_p^*\text{.}\) Thus, if \(k\nmid \phi(p)\text{,}\) then there is no element \(r\in (\Z/p\Z)^*\) of order \(k\text{.}\) Equivalently, \(F_k=\nonempty\) for all \(k\nmid \phi(p)\text{.}\)
    🔗
    🔗
  2. For each positive divisor \(d\) of \(\phi(p)\) we know from Lemma 1.3.6 that
    \begin{align*} \abs{F_d} \amp \in \{0,\phi(d)\}\text{.} \end{align*}
    In particular, we have \(\abs{F_d}\leq \phi(d)\) for all \(d\mid \phi(p)\text{.}\) Furthermore, since the sets \(F_d\) are disjoint (an element of \((\Z/p\Z)^*\) has a unique order), and since \((\Z/p\Z)^*=\bigcup_{d\mid n}F_d\) (every element of \((\Z/p\Z)^*\) has order \(d\) for some divisor \(d\) of \(\phi(p)\)), we have
    \begin{align} p-1 \amp =\abs{\F_p^*}=\sum_{d\mid p-1}\abs{F_d} \text{.}\tag{1.3.6} \end{align}
    From Lemma 1.3.5 we also know that
    \begin{align*} p-1 \amp = \sum_{d\mid p-1}\phi(d)\text{.} \end{align*}
    Subtracting these two equalities, we conclude that
    \begin{align*} 0 \amp =\sum_{d\mid p-1}(\phi(d)-\abs{F_d})\text{.} \end{align*}
    Since \(\abs{F_d}\leq \phi(d)\) for all positive \(d\mid p-1\text{,}\) we conclude that \(\phi(d)-\abs{F_d}\geq 0\) for all positive \(d\mid p-1\text{.}\) But a sum of nonnegative numbers is zero if and only if each number is zero. Thus we conclude that
    \begin{align*} \abs{F_d} \amp = \phi(d) \end{align*}
    for all positive \(d\mid p-1\text{.}\)
    🔗
    🔗
  3. From (2), we conclude that \(\ans{F_{p-1}}=\phi(p-1)\text{.}\) Since \(\phi(p-1)\) is positive, we conclude that there is an element \(r\in \F_p^*\) of order \(p-1\text{.}\) (In fact there are precisely \(\phi(p-1)\) such elements.) Such an element satisfies \(\angvec{r}=\F_p^*\) and hence is a primitive root.
