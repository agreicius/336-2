Sage example 3. Primitive roots.

The Sage function primitive_root(n) returns a primitive root of \((\Z/n\Z)^*\) if one exists, and raises an error otherwise. Furthermore, if \(n=p\) is prime, then primitive_root(p) returns the smallest positive integer \(a\) such that \([a]_p\) is a primitive root of \((\Z/p\Z)^*\text{.}\)
