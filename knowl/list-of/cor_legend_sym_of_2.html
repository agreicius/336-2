Corollary 1.6.6. Formula for \(\qr{2}{p}\).

Let \(p\) be an odd prime. We have
\begin{equation} \qr{2}{p}=\begin{cases} 1 \amp \text{if } p\equiv \pm 1 \pmod 8 \\ -1 \amp \text{if } p\equiv \pm 3 \pmod 8 \text{.}\end{cases}\tag{1.6.7} \end{equation}
🔗
🔗

Proof.

Let
\begin{align} X \amp =\{[1]_p, \dots [(p-1)/2]_p\}\tag{1.6.8}\\ Y \amp =\{[(p+1)/2]_p,\dots ,[p-1]_p\}\text{.}\tag{1.6.9} \end{align}
Using Theorem 1.6.1, we have
\begin{equation*} \qr{2}{p}=(-1)^n\text{,} \end{equation*}
where \(n\) is the number of multiples of \([2k]_p\text{,}\) \(1\leq k\leq (p-1)/2\text{,}\) that lie in \(Y\text{.}\)
🔗
We treat two separate cases: \(p\equiv 1\pmod 4\) and \(p\equiv -1\pmod 4\text{.}\)
🔗

Case: \(p\equiv 1 \pmod 4\).

Assume \(p\equiv 1\pmod 4\text{,}\) or equivalently, \(p-1=4k\) for some integer \(k\text{.}\) In this case we have \((p-1)/2=2k\text{,}\) which is divisible by \(2\text{.}\) It follows that there are exactly \((p-1)/4\) multiples of \([2]_p\) in \(X\) and \(Y\text{,}\) and thus that
\begin{equation*} \qr{2}{p}=(-1)^{(p-1)/4}\text{.} \end{equation*}
Since \((p-1)/4=k\in \Z\text{,}\) we see that this exponent is even if and only if \(2\mid k\text{,}\) and it is easy to see that this is true if and only if \(p\equiv 1\pmod 8\text{.}\) Since \(p\equiv 1\pmod 4\) implies \(p\equiv 1 \pmod 8\) or \(p\equiv 5 \pmod 8\text{,}\) we conclude that in this case we have
\begin{equation*} \qr{2}{8}=\begin{cases} 1 \amp \text{if } p\equiv 1\pmod 8\\ -1 \amp \text{if } p\equiv 5\pmod 8 \text{.}\end{cases} \end{equation*}
🔗
🔗

Case: \(p\equiv 3\pmod 4\).

Assume \(p\equiv 3\pmod 4\text{,}\) or equivalently, \(p=4k+3\) for some integer \(k\text{.}\) In this case \((p-3)/2\) is a multiple of \(2\text{,}\) but \((p-1)/2\) is not. It follows that there are \((p-3)/4\) multiples os \([2]_p\) lying in \(X\) and \((p+1)/4\) multiples of \([2]_p\) lying in \(Y\text{.}\) We conclude that
\begin{equation*} \qr{2}{p}=(-1)^{(p+1)/4}\text{.} \end{equation*}
Since \((p+1)/4=(k+1)\text{,}\) we see that this exponent is even if and only if \(k\) is odd, which is true if and only if \(p\equiv 7 \pmod 8\text{.}\) Since \(p\equiv 3\pmod 4\) implies \(p\equiv 3 \pmod 8\) or \(p\equiv 7 \pmod 8\text{,}\) we conclude in this case that
\begin{equation*} \qr{2}{8}=\begin{cases} 1 \amp \text{if } p\equiv 7\pmod 8\\ -1 \amp \text{if } p\equiv 3 \pmod 8 \text{.}\end{cases} \end{equation*}
🔗
🔗
Since our two cases are exhaustive, we combine them to conclude
\begin{equation*} \qr{2}{8}=\begin{cases} 1 \amp \text{if } p\equiv 1\pmod 8 \\ -1\amp \text{if } p\equiv 5\pmod 8\\ 1 \amp \text{if } p\equiv 7\pmod 8\\ -1\amp \text{if } p\equiv 3\pmod 8 \text{,}\end{cases} \end{equation*}
or more succinctly,
\begin{equation*} \qr{2}{8}=\begin{cases} 1 \amp \text{if } p\equiv \pm 1\pmod 8\\ -1\amp \text{if } p\equiv \pm 3\pmod 8 \text{.}\end{cases} \end{equation*}
