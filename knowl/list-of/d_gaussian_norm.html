Definition 1.11.4. Gaussian norm.

The norm function on \(\Z[i]\) is the function \(N\colon \Z[i]\rightarrow \Z_{\geq 0}\) defined as \(N(\alpha)=\abs{\alpha}^2\text{.}\) Equivalently, if \(\alpha=a+ib\text{,}\) we have
\begin{align} N(\alpha) \amp = \alpha\cdot \overline{\alpha}=a^2+b^2\text{.}\tag{1.11.2} \end{align}
