    Section 1.8 Pseudoprimes

    Subsection 1.8.1 Jacobi symbol

    Definition 1.8.1. Jacobi symbol.

    Let \(n\) be an odd positive integer with prime factorization \(\prod_{i=1}^r p_i^{n_i}\text{.}\) Given any integer \(a\text{,}\) we define the Jacobi symbol \(\qr{a}{n}\) as
    \begin{equation} \qr{a}{n}=\prod_{i=1}^r \qr{a}{p_i}^{n_i}.\tag{1.8.1} \end{equation}
    🔗
    🔗

    Remark 1.8.2. Jacobi symbol and squares modulo \(n\).

    As we will see below, the Jacobi symbol is a convenient extension of the the Legendre symbol to odd composite moduli that satisfies many of the same algebraic properties. We hasten to add, however, that when \(n\) is not prime, the Jacobi symbol cannot be used to determine whether a given integer \(a\) is a square modulo \(n\text{;}\) that is it is not the case that \(\qr{a}{n}=1\) if and only if \(a\) is a square modulo \(n\) for integers \(a\) relatively prime to \(n\text{.}\) Consider \(\qr{2}{15}\text{,}\) for example. We have
    \begin{align*} \qr{2}{15} \amp =\qr{2}{3}\qr{2}{5}\\ \amp = (-1)(-1)=1\text{,} \end{align*}
    and yet \(2\) is not a square modulo \(15\text{,}\) as is easily verified.
    🔗
    However, it is possible to show that if \(\qr{a}{n}=-1\text{,}\) then \(a\) is not a square modulo \(n\text{.}\) (Hint: CRT!)
    🔗
    🔗

    Proposition 1.8.3. Jacobi symbol.

    Assume \(n\) is an odd positive integer.
    1. If \(a\equiv b\pmod n\text{,}\) then \(\qr{a}{n}=\qr{b}{n}\text{.}\)
      🔗
      🔗
    2. \(\qr{ab}{n}=\qr{a}{n}\qr{b}{n}\) for all integers \(a\) and \(b\text{.}\)
      🔗
      🔗
    3. Given odd positive integer \(m\text{,}\) we have
      \begin{align} \qr{a}{mn} \amp =\qr{a}{m}\qr{a}{n}\tag{1.8.2} \end{align}
      for all integers \(a\text{.}\)
      🔗
      🔗
    4. \begin{align} \qr{-1}{n} \amp = \begin{cases} 1 \amp \text{if } n \equiv 1 \pmod{4} \\ -1 \amp \text{if } n \equiv 3 \pmod{4} \end{cases}\tag{1.8.3}\\ \qr{2}{n} \amp = \begin{cases} 1 \amp \text{if } n \equiv \pm 1 \pmod{8} \\ -1 \amp \text{if } n \equiv \pm 3\pmod{8} \text{.}\end{cases}\tag{1.8.4} \end{align}
      🔗
    5. Let \(m\) be an odd positive integer. We have
      \begin{align} \qr{m}{n} \amp =(-1)^{\frac{m-1}{2}\cdot \frac{n-1}{2}}\qr{n}{m}\text{.}\tag{1.8.5} \end{align}
      🔗
      🔗
    🔗
    🔗
    🔗

    Subsection 1.8.2 Pseudoprimes

    Definition 1.8.4. Pseudoprimes.

    Let \(n\) be an odd positive integer and let \(a\) be an integer that is relatively prime to \(n\text{.}\)
    • We say \(n\) is a pseudoprime to the base \(a\) if \(a^{n-1}\equiv 1\pmod n\text{.}\)
      🔗
      🔗
    • We say that \(n\) is an Euler pseudoprime to the base \(a\) if
      \begin{align} a^{(n-1)/2} \amp \equiv \qr{a}{n} \pmod n\text{.}\tag{1.8.6} \end{align}
      🔗
      🔗
    • Let \(n-1=2^km\text{,}\) where \(m\) is odd. We say \(n\) is a strong pseudoprime wto the base \(a\) if one of the following conditions is satisfied:
      1. \(a^m\equiv 1\pmod n\text{;}\)
        🔗
        🔗
      2. \(a^{2^\ell}\equiv -1\pmod n\) for some \(1\leq \ell \leq k-1\text{.}\)
        🔗
        🔗
      When one of the above conditions is satisfied, we say that \(n\) passes Miller’s test for the base \(b\).
      🔗
      🔗
    🔗
    🔗

    Theorem 1.8.5. Pseudoprimes:.

    1. Using logical shorthand:
      \begin{equation} \text{ strong pseudoprime} \implies \text{ Euler pseudoprime} \implies \text{ pseudoprime}\text{.}\tag{1.8.7} \end{equation}
      🔗
      🔗
    2. If \(p\) is an odd prime, then \(p\) passes Miller’s test with base \(a\) for all \(1\leq a\leq p-1\text{.}\)
      🔗
      🔗
    3. If \(n\) is a positive composite integer, then \(n\) is a strong pseudoprime to the base \(a\) for at most \(1/4\) of the integers \(a\) in \(\{1,2,\dots, n-1\}\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.

    1. Left to the reader.
      🔗
      🔗
    2. Left to the reader.
      🔗
      🔗
    3. See Section 9.4 of Rosen’s text for a proof. This is essentially just an application of Theorem 1.4.1 and the Chinese remainder theorem. In this sense the ideas behind the proof will be familiar enough to you. The necessary inequality calisthenics, however, make the proof somewhat lengthy.
      🔗
      🔗
    🔗
    🔗

    Procedure 1.8.6. Miller-Rabin probabilistic primality test.

    Let \(n\) be an odd positive integer, and let \((a_1,a_2,\dots, a_k)\) be a sequence of \(k\) elements chosen randomly from \(\{1,2,\dots, n-1\}\text{.}\) If \(n\) is composite, then the probability that \(n\) passes Miller’s test for all of the bases \(a_1,a_2,\dots, a_k\) is at most \((1/4)^k\text{.}\)
    🔗
    🔗

    Remark 1.8.7. Strong pseudoprimes.

    1. There are infinitely many strong pseudoprimes (to some base). In fact, you can show that there are infinitely many strong pseudoprimes to the base \(2\text{.}\)
      🔗
      🔗
    2. The following observations imply that for reasonably sized integers \(n\text{,}\) applying Miller’s test to \(2\text{,}\) \(3\text{,}\) \(5\text{,}\) and \(7\) is sufficient to determine whether \(n\) is prime.
      • \(2047\) is the smallest positive integer that is a strong pseudoprime to the base \(2\text{.}\)
        🔗
        🔗
      • \(1,373,653\) is the smallest positive integer that is a strong pseudoprime to the bases \(2\) and \(3\text{.}\)
        🔗
        🔗
      • \(25,326,001\) is the smallest positive integer that is a strong pseudoprime to the bases \(2\text{,}\) \(3\text{,}\) and \(5\text{.}\)
        🔗
        🔗
      • \(3,215,031,751\) is the smallest positive integer that is a strong pseudoprime to the bases \(2\text{,}\) \(3\text{,}\) \(5\text{,}\) and \(7\text{.}\)
        🔗
        🔗
      🔗
      🔗
