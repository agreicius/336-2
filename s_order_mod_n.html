      🔗

    Subsection 1.2.1 Order and primitive roots

    Recall Euler’s theorem: if \(n\) is a positive integer and \(a\in \Z\) is relatively prime to \(n\text{,}\) then \(a^{\phi(n)}\equiv 1 \pmod n\text{.}\) Equivalently, in the language of modular rings, given \(r=[a]_n\in (\Z/n\Z)^*\text{,}\) we have
    \begin{gather*} r^{\phi(n)}=[a]_n^{\phi(n)}=[a^{\phi(n)}]=1\text{.} \end{gather*}
    As an illustration of the economy of the modular ring language, let’s re-prove this result.
    🔗

    Modular ring proof of Euler’s theorem.

    Let \(n\) be a positive element, and let \(r_1,r_2,\dots, r_{\phi(n)}\) be the distinct elements of \((\Z/n\Z)^*\text{,}\) so that
    \begin{align*} (\Z/n\Z)^* \amp = \{r_1,r_2,\dots, r_{\phi(n)}\}\text{.} \end{align*}
    (Note: although we have \(r_i=[a_i]_n\) for some integer \(a_i\text{,}\) we will not need to make use of these integers in our argument; we will only use properties of \((\Z/n\Z)^*\) discussed in the previous section.)
    🔗
    Now, given any element \(r\in (\Z/n\Z)^*\text{,}\) we claim that we have the set equality
    \begin{align*} \{r_1,r_2,\dots, r_{\phi(n)}\} \amp = \{r\,r_1, r\, r_{2},\dots, r\, r_{\phi(n)}\}\text{.} \end{align*}
    Indeed, from Theorem 1.1.15, we know that \(r\, r_i\in (\Z/n\Z)^*\text{,}\) since \((\Z/n\Z)^*\) is closed under multiplication. Thus
    \begin{align*} \{r\,r_1, r\,r_2,\dots, r\,r_{\phi(n)}\} \amp\subseteq (\Z/n\Z)^*=\{r_1,r_2,\dots, r_{\phi(n)}\} \text{.} \end{align*}
    For the other inclusion, we use the fact that there is an element \(r^{-1}\in (\Z/n\Z)^*\) satisfying \(r\,r^{-1}=1\text{.}\) Given any element \(r_i\in (\Z/n\Z)^*\text{,}\) letting \(r_j=r^{-1}\cdot r_i\text{,}\) we have
    \begin{align*} r\,r_j \amp = r(r^{-1}r_i)\\ \amp = (rr^{-1})r_i\\ \amp = 1r_i\\ \amp = r_i\text{.} \end{align*}
    Having shown \(\{r_1,r_2,\dots, r_{\phi(n)}\}=\{rr_1,rr_2,\dots, rr_{\phi(n)}\}\text{,}\) we see that
    \begin{align*} \prod_{i=1}^{\phi(n)}r_i \amp =\prod_{i=1}^{\phi(n)}rr_i\\ \amp = r^{\phi(n)}\prod_{i=1}^{\phi(n)}r_i\text{.} \end{align*}
    Letting \(s=\prod_{i=1}^{\phi(n)}r_i\text{,}\) the above equality becomes
    \begin{align*} s \amp =r^{\phi(n)}s\text{.} \end{align*}
    Since \(s\in (\Z/n\Z)^*\text{,}\) it can be cancelled multiplicatively by multiplying both sides by \(s^{-1}\text{.}\) We conclude that \(1=r^{\phi(n)}\text{,}\) as desired.
    🔗
    🔗
    From Euler’s theorem we know that for all \(r\in (\Z/n\Z)^*\) there is at least one positive integer \(k\) such that \(r^k=1\text{:}\) namely, \(k=\phi(n)\text{.}\) It follows from the well-ordering property of the integers that there is thus a smallest positive integer \(k\) satisfying \(r^k=1\text{.}\) We call this the order of \(r\text{.}\)
    🔗

    Definition 1.2.1. Order of a modular unit.

    Let \(n\) be a positive integer. Given an element \(r\in (\Z/n\Z)^*\text{,}\) its order is the smallest positive integer \(k\) such that \(r^k=1\text{.}\)
    🔗
    Similarly, given an integer \(a\in \Z\) that is relatively prime to \(n\text{,}\) the order of \(a\) modulo \(n\), denoted \(\ord_n a\text{,}\) is the smallest positive integer \(k\) such that \(a^k\equiv 1\pmod n\text{.}\)
    🔗
    🔗

    Remark 1.2.2. Modular exponentiation.

    In what follows we will take for granted some elementary algebraic properties about exponentiation in \(\Z/n\Z\) that follow immediately from the corresponding properties in \(\Z\text{.}\) For example, we have
    \begin{align*} r^{j+k} \amp = r^jr^k \amp r^{jk}=(r^j)^k=(r^k)^j \end{align*}
    for all \(r\in \Z/n\Z\) and nonnegative integers \(j,k\in \Z_{\geq 0}\text{.}\) Furthermore, if \(r\in (\Z/n\Z)^*\text{,}\) then definining negative powers as \(r^{-n}=(r^{-1})^n\text{,}\) the above equalities hold for all integers \(j,k\in \Z\text{.}\)
    🔗
    🔗

    Proposition 1.2.3. Order of modular units.

    Let \(n\) be a positive integer, let \(r\in (\Z/n\Z)^*\text{,}\) and let \(m=\ord r\text{.}\)
    1. For all integers \(k\text{,}\) \(r^k=1\) if and only if \(m\mid k\text{.}\)
      🔗
      In particular \(\ord r\mid \phi(n)\text{.}\)
      🔗
      🔗
    2. \(r^i=r^j\) if and only if \(i\equiv j\pmod m\text{.}\)
      🔗
      🔗
    3. We have
      \begin{align*} \{r^i\mid i\in \Z\} \amp = \{1,r,r^2,\dots, r^{m-1}\}\text{,} \end{align*}
      and this set has cardinality \(m\) (i.e., it contains exactly \(m\) distinct elements).
      🔗
      🔗
    4. \(\ord r^i=m/\gcd(i,m)\) for all integers \(i\in \Z\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.

    1. First assume that \(m\mid k\text{,}\) so that \(k=mq\) for some \(q\in \Z\text{.}\) We then have
      \begin{align*} r^k \amp =r^{mk}\\ \amp = (r^m)^k\\ \amp =1^k=1\text{,} \end{align*}
      as desired. Conversely, assume \(r^k=1\) and write \(k=qm+s\) where \(0\leq s < m\text{,}\) as in the division algorithm. Since
      \begin{align*} 0 \amp =r^{k}\\ \amp = r^{qm+s}\\ \amp = r^{qm}r^s\\ \amp =r^s \amp (m\mid qm)\text{,} \end{align*}
      and since \(m\) is the smallest positive integer satisfying \(r^m=1\text{,}\) we must have \(s=0\text{,}\) and thus \(m\mid k\text{.}\)
      🔗
      🔗
    2. We have
      \begin{align} r^i=r^j \amp \iff r^{i}r^{-j}=r^{j}r^{-j}\tag{1.2.1}\\ \amp \iff r^{i-j}=1\tag{1.2.2}\\ \amp \iff m\mid (i-j) \amp (\text{by } (1))\tag{1.2.3}\\ \amp \iff i\equiv j\pmod m\text{.}\tag{1.2.4} \end{align}
      🔗
      🔗
    3. Since \(r^i=r^j\) if and only if \(i\equiv j\pmod m\text{,}\) and since \(\{0,1,\dots, m-1\}\) form a complete set of residues modulo \(m\text{,}\) we have
      \begin{align} \{r^i\mid i\in \Z\} \amp = \{r^0, r^1,\dots, r^{m-1}\} \tag{1.2.5}\\ \amp =\{1,r,\dots, r^{m-1}\}\text{,}\tag{1.2.6} \end{align}
      and these powers of \(r\) are all distinct. Thus the set has cardinality \(m\text{.}\)
      🔗
      🔗
    4. Let \(k=m/\gcd(i,m)\text{.}\) We will show that \(k\) is the smallest positive integer such that \((r^i)^k=1\text{,}\) and thus that \(\ord r^i=k\text{.}\)
      🔗
      First observe that \((r^{i})^k=r^{ik}=r^{(i/\gcd(i,m))m}=1\text{,}\) since \(m\mid (i/\gcd(i,m))m\text{.}\) Now assume that \(\ell\) is a positive integer satisfying \((r^i)^\ell=r^{i\ell}=1\text{.}\) It follows that \(m\mid i\ell\text{.}\) But then
      \begin{align*} m/\gcd(i,m) \amp \mid (i/\gcd(i,m))\ell\text{.} \end{align*}
      Since \(m/\gcd(i,m)\) and \(i/\gcd(i,m)\) are relatively prime, it follows that \(k=m/\gcd(i,m)\mid \ell\text{,}\) and hence that \(k\leq \ell\text{,}\) since \(k\) and \(\ell\) are positive. This proves \(k\) is the smallest positive integer such that \((r^i)^k=1\text{,}\) and thus that \(\ord r^i=k\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Definition 1.2.4. Primitive root.

    Let \(n\) be a positive integer, and let \(r\in (\Z/n\Z)^*\text{.}\) The cyclic group generated by \(r\), denoted \(\langle r\rangle\text{,}\) is the set of all integer powers of \(r\text{:}\) i.e.,
    \begin{align*} \angvec{r} \amp = \{r^i\mid i\in \Z\}\\ \amp = \{1,r, \dots, r^{m-1}\}\text{,} \end{align*}
    where \(m=\ord r\text{.}\)
    🔗
    A primitive root (or cyclic generator) of \((\Z/n\Z)^*\) is an element \(r\in (\Z/n\Z)^*\) satisfying \(\angvec{r}=(\Z/n\Z)^*\text{.}\)
    🔗
    We say an integer \(a\) is a primitive root modulo \(n\) if \(r=[a]_n\) is a primitive root of \((\Z/n\Z)^*\text{.}\)
    🔗
    🔗

    Example 1.2.5. Primitive roots.

    Compute the order of all elements of \((\Z/n\Z)^*\) for the given modulus \(n\text{,}\) and decide whether \((\Z/n\Z)^*\) has a primitive root.
    🔗
    Additionally, if a primitive root \(r\in (\Z/n\Z)^*\) does exist, verify the order formula for each power \(r^i\) given in Proposition 1.2.3.
    1. \(\displaystyle n=7\)
      🔗
      🔗
    2. \(\displaystyle n=12\)
      🔗
      🔗
    3. \(\displaystyle n=25\)
      🔗
      🔗
    🔗
    Solution.
    For each modulus below, we will drop most of our notational baggage and simply write \(a\) instead of \([a]_n\text{.}\)
    1. We have \((\Z/7\Z)^*=\{1,2,3,4,5,6\}\text{.}\) Computing successive powers of these elements leads to the following order computations:
      \begin{align*} \ord 1 \amp =1 \amp \ord 2=3\amp \ord 3\amp =6\\ \ord 4 \amp = 3 \amp \ord 5\amp =6 \amp \ord 6\amp =2\text{.} \end{align*}
      Since \(\ord 3=\ord 5=6\text{,}\) we see that both of these elements are primitive roots in \(\Z/7\Z\text{.}\) Let’s verify that \(\angvec{3}=\angvec{5}=(\Z/7\Z)^*\text{:}\)
      \begin{align*} \angvec{3} \amp =\{1, 3, 3^2, 3^3, 3^4, 3^5, 3^6\}\\ \amp = \{1,3,2,6, 4,5\}=(\Z/7\Z)^*\\ \angvec{5} \amp = \{1, 5, 5^2, 5^3, 5^4, 5^5, 5^6\} \\ \amp = \{1,5,4,6,2,3\}=(\Z/7\Z)^*\text{.} \end{align*}
      Lastly, let’s verify the order formula for the powers of \(3\text{.}\)
      🔗
      🔗
    2. We have \((\Z/12\Z)^*=\{1,5,7,11\}\) and we easily compute
      \begin{align*} \ord 1 \amp = 1 \amp \ord 5\amp=2\\ \ord 7 \amp =2 \amp \ord 11\amp=2\text{.} \end{align*}
      Since no element has order 4, there is no primitive root for \((\Z/12\Z)^*\text{.}\)
      🔗
      🔗
    3. Since \(\gcd(a,25)=1\) if and only if \(5\nmid a\text{,}\) the units of \(\Z/25\Z\) are the elements of the form \([a]_{25}\text{,}\) where \(5\nmid a\text{.}\) There are \(20=25-5\) of these. Let’s use Sage to see whether any of these have order 20.
      🔗
      We see that \(2\) has order \(20=\abs{(Z/25\Z)^*)}\text{,}\) and thus that \(2\) is a primitive root. We leave it to the reader to verify the order formula for powers of \(2\) as we did in the first part.
      🔗
      On a separate note, notice that there are 8 distinct primitive roots here: \(2,3,8,12,13,17,22,23\text{.}\) In the next sections we will derive a formula for the number of distinct primitive roots modulo, if one does indeed exist.
      🔗
      🔗
    🔗
    🔗
    🔗
    Our example makes one thing clear: some moduli admit primitive roots; others do not. In the next section we will see that when \(n=p\) is prime, then a primitive root is guaranteed to exist.
    🔗

    Subsection 1.2.2 Sage examples

    Here are some Sage cells to play with.
    🔗

    Sage example 2. Order and primitive elements.

    For an element a of Integers(n), the method a.is_primitive_root() determines whether a is a primitive root.
    🔗
    For an invertible element a of Integers(n), the method a.multiplicative_order() computes the order of a. If a is not invertible, then a.multiplicative_order() raises an error. We can test for invertibility using the a.is_unit() method.
    🔗
    Lastly, a modular ring object R=Integers(n) has a method R.unit_group() that returns \((\Z/n\Z)^*\text{,}\) which itself has lots of useful methods that you can explore using dir(G). The method G.is_cyclic() determines whether \((\Z/n\Z)^*\) has a primitive element. (Terminology-wise, \((\Z/n\Z)^*\) is a mathematical object called a group; and a group is cyclic precisely when it can be expressed as \(\angvec{r}\) for some element \(r\text{,}\) just as in Definition 1.2.4.)
