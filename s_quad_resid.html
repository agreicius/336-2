    Section 1.5 Quadratic residues and \(n\)-th powers

    Objectives

    1. Describe the \(n\)-th powers of \(\Z/p\Z\) in terms of a primitive root modulo \(p\text{.}\)
      🔗
      🔗
    2. Define the Legendre symbol and derive some preliminary properties.
      🔗
      🔗
    🔗

    Subsection 1.5.1 The \(n\)-th powers of \(\Z/p\Z\)

    Definition 1.5.1. \(n\)-th power.

    Let \(n\) be a positive integer, and let \(p\) be a prime. An element \(r\in \F_p\) is an \(n\)-th power if there is an \(s\in \F_p\) satisfying \(s^n=r\text{.}\) We call \(s\) an \(n\)-th root of \(r\) in this case.
    🔗
    Similarly, we say an integer \(a\in \Z\) is an \(n\)-th power modulo \(p\) if \([a]_p\) is an \(n\)-th power in \(\F_p\text{:}\) equivalently, if there is an integer \(b\in \Z\) such that \(b^n\equiv a \pmod p\text{.}\) We call \(b\) an \(n\)-th root of \(a\) modulo \(p\) in this case.
    🔗
    🔗

    Theorem 1.5.2. \(n\)-th powers modulo \(p\).

    Let \(p\) be a prime, let \(r\in \F_p^*\) be a primitive root, and let \(n\) be a positive integer. Define \(S_n\) to be the set of all \(n\)-th powers in \(\F_p^*\text{,}\) and let \(d=\gcd(n,p-1)\)
    1. For all \(i\in \Z\text{,}\) we have \(r^i\in S_n\) if and only if \(d\mid i\text{.}\)
      🔗
      🔗
    2. Let \(f(x)=x^{(p-1)/d}-1\in \F_p[x]\text{.}\) We have
      \begin{align} S_n \amp = \angvec{r^d}= \{s\in \F_p\mid f(s)=0\}\text{.}\tag{1.5.1} \end{align}
      In particular, we have \(s\in S_n\) if and only if \(s^{(p-1)/d}=1\text{;}\) equivalently, \(a\in \Z\) is an \(n\)-th power modulo \(p\) if and only if \(a^{(p-1)/d}\equiv 1\pmod p\text{.}\)
      🔗
      🔗
    3. \(\abs{S_n}=(p-1)/\gcd(n,p-1)\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.

    1. Any element \(s\in \F_p^*\) can be written as \(s=r^i\) for some \(i\in \Z\text{.}\) We have \(s\in S_n\) if and only if \(s=t^n\) for some \(t=r^j\text{:}\) or equivalently, if and only if \(r^i=(r^j)^n=r^{nj}\) for some \(j\in \Z\text{.}\) Since \(\ord r=p-1\text{,}\) we have we have \(r^i=r^{nj}\) if and only if \(i\equiv nj \pmod {p-1}\text{.}\) Thus \(s=r^i\) is an \(n\)-th power if and only if there exists \(j\) such that \(i\equiv nj \pmod {p-1}\text{.}\) From congruence theory from the previous quarter, we know this is true if and only if \(d=\gcd(n,p-1)\mid i\text{.}\)
      🔗
      🔗
    2. Since by the first part we know that \(s=r^i\in S_n\) if and only if \(d\mid i\text{,}\) to prove the first equality, it suffices to show that \(\angvec{r^d}=\{r^i\mid i\in \Z,d\mid i\}\text{.}\) If \(d\mid i\text{,}\) then \(i=qd\) for some \(q\in \Z\text{,}\) and \(r^i=(r^d)^q\in \angvec{r^d}\text{.}\) For the other inclusion, if \(r^i\in \angvec{r^d}\text{,}\) then \(r^i=(r^d)^j=r^{dj}\) for some \(j\text{,}\) and thus \(i\equiv dj \pmod {p-1}\text{.}\) Since \(d\mid dj\) and \(d\mid p-1\text{,}\) it follows that \(d\mid i\text{,}\) as desired.
      🔗
      As for the second equality, since any element \(s\in \angvec{r^d}\) satisfies \(s^{(p-1)/d}=1\text{,}\) we have \(\angvec{r^d}\subseteq \{s\in \F_p\mid f(s)=0\}\text{.}\) Since, furthermore, the polynomial \(f(x)=x^{(p-1)/d}-1\) has at most \((p-1)/d\) roots and \(\abs{\angvec{r^d}}=\ord r^d=(p-1)/d\text{,}\) we conclude that \(\angvec{r^d}=\{s\in \F_p\mid f(s)=0\}\text{.}\)
      🔗
      🔗
    3. We have
      \begin{align*} \abs{S_n} \amp =\abs{\angvec{r^d}}=\ord r^d=(p-1)/d\text{.} \end{align*}
      🔗
      🔗
    🔗
    🔗

    Corollary 1.5.3. Squares modulo \(p\).

    Let \(p\) be an odd prime. There are exactly \((p-1)/2\) squares in \(\F_p^*\text{.}\) In other words, half of the elements of \(\F_p^*\) have square roots in \(\F_p^*\text{.}\)
    🔗
    🔗

    Proof.

    This is the special case \(n=2\) of Theorem 1.5.2.
    🔗
    🔗

    Example 1.5.4. \(n\)-th powers modulo \(13\).

    Let \(p=13\text{.}\) For each positive divisor \(n\) of \(\phi(p)\text{,}\) determine the set \(S_n\) of \(n\)-th powers in \(\F_{13}^*\text{.}\)
    🔗
    Solution.
    The divisors of \(\phi(13)=12\) are \(1,2,3,4,6,12\text{.}\) The element \([2]_{13}\) is a primitive root modulo \(13\text{.}\) We will write \(2=[2]_{13}\) for simplicity. We have
    \begin{align*} S_{1} \amp = \angvec{2}=\F_{13}^* \amp \\ S_2\amp =\angvec{2^2}=\{4,4^2=3,4^3=12,4^4=9,4^5=10,4^6=1\} \amp \\ S_{3} \amp = \angvec{2^3}=\{8, 8^2=12,8^3=5, 8^4=1\} \amp \\ S_{4} \amp = \angvec{2^4}=\{3,3^2=9, 3^3=1\} \amp \\ S_{6} \amp = \angvec{2^6}=\{12,12^2=1\} \amp \\ S_{12} \amp = \angvec{2^{12}}=\{1\}\amp \text{.} \end{align*}
    Let’s confirm our answer with Sage.
    🔗
    🔗
    🔗
    🔗

    Subsection 1.5.2 Quadratic residues

    Definition 1.5.5. Quadratic residue.

    Let \(p\) be an odd prime. An element \(r\in \F_p^*\) is a quadratic residue if \(r\) is a square in \(\F_p^*\) (equivalently, if \(r\) has a square root in \(\F_p^*\)). Similarly, an integer \(a\in \Z\) with \(p\nmid a\) is a quadratic residue modulo \(p\) if \(a\) is a square modulo \(p\text{.}\)
    🔗
    🔗

    Definition 1.5.6. Legendre symbol.

    Let \(p\) be an odd prime. Given an integer \(a\in \Z\text{,}\) we define the Legendre symbol \(\qr{a}{p}\) as
    \begin{equation*} \qr{a}{p}=\begin{cases} 0 \amp \text{if } p\mid a \\ 1 \amp \text{if } p\nmid a \text{ and } a \text{ is a square modulo } p\\ -1 \amp \text{if } p\nmid a \text{ and } a \text{ is not a square modulo } p \text{.}\end{cases} \end{equation*}
    The Legendre symbol (with respect to \(p\)) gives rise to a function \(f\colon \Z\rightarrow \{\pm 1\}\) defined as \(f(a)=\qr{a}{p}\text{.}\)
    🔗
    🔗

    Theorem 1.5.7. Quadratic residues.

    Let \(p\) be an odd prime.
    1. If \(a\equiv b\pmod p\text{,}\) then \(\qr{a}{p}=\qr{b}{p}\text{.}\)
      🔗
      🔗
    2. \(\qr{ab}{p}=\qr{a}{p}\qr{b}{p}\) for all \(a,b\in \Z\text{.}\)
      🔗
      🔗
    3. \(\qr{a^2}{p}=1\) for all \(a\in \Z\) with \(p\nmid a\text{.}\)
      🔗
      🔗
    4. Exactly \((p-1)/2\) (i.e., half) of the elements of \(\{1,2,\dots, p-1\}\) are quadratic residues modulo \(p\text{.}\)
      🔗
      🔗
    5. Euler’s criterion.
      \(\qr{a}{p}\equiv a^{(p-1)/2}\pmod p \) for all \(a\in \Z\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.

    1. This is essentially by definition and the observation that \(\qr{a}{p}=0\) if and only if \(p\mid a\text{,}\) if and only if \(a\equiv 0\pmod p\text{.}\)
      🔗
      🔗
    2. It is clear that if either of \(a\) and \(b\) are divisible by \(p\text{,}\) then \(0=\qr{ab}{p}=\qr{a}{p}\qr{b}{p}\text{.}\) Assume now that \(a\) and \(b\) are relatively prime to \(p\text{.}\) Let \(r\) be a primitive root modulo \(p\text{,}\) and let \([a]_p=r^i\) and \([b]_p=r^j\text{.}\) We know that an element \(r^k\in \F_p^*\) is a square if and only if \(k\) is even. It follows that \(\qr{a}{p}=(-1)^i\text{,}\) \(\qr{b}{p}=(-1)^j\text{.}\) Furthermore, since \([ab]_p=r^{i+j}\text{,}\) we have \(\qr{ab}{p}=(-1)^{i+j}\text{.}\) We conclude that
      \begin{align*} \qr{a}{p}\qr{b}{p} \amp = (-1)^i\cdot (-1)^j\\ \amp = (-1)^{i+j}\\ \amp = \qr{ab}{p}\text{.} \end{align*}
      🔗
      🔗
    3. We have \(\qr{a^2}{p}=\qr{a}{p}\qr{a}{p}=\left(\qr{a}{p}\right)^2=1\text{,}\) since \(\qr{a}{p}=\pm 1\text{.}\)
      🔗
      🔗
    4. Let \(s=[a]_p\text{.}\) By Theorem 1.5.2, we know that \(s\) is a square if and only if \(s^{(p-1)/2}=1\text{.}\) Since \(x=s^{(p-1)/2}\) satisfies \(x^2=1\text{,}\) and since the only roots of \(f(x)=x^2-1\) are \([1]_p\) and \([-1]_p\text{,}\) we see that either \(a^{(p-1)/2}\equiv 1\pmod p\) and \(\qr{a}{p}=1\text{,}\) or \(a^{(p-1)/2}\equiv -1\pmod p\) and \(\qr{a}{p}=-1\text{.}\) In both cases we have \(a^{(p-1)/2}\equiv \qr{a}{p} \pmod p\text{,}\) as desired.
      🔗
      🔗
    🔗
    🔗

    Corollary 1.5.8. Sum of Legendre symbols.

    Let \(p\) be an odd prime. We have
    \begin{align*} \sum_{i=1}^{p-1}\qr{i}{p} \amp = 0\text{.} \end{align*}
    🔗
    🔗

    Corollary 1.5.9. Square roots of \(-1\) modulo \(p\).

    Let \(p\) be an odd prime. We have
    \begin{align} \qr{-1}{p} \amp =(-1)^{(p-1)/2}\tag{1.5.2}\\ \amp =\begin{cases} 1\amp \text{if } p\equiv 1\pmod 4 \\ -1\amp \text{if } p\equiv 3\pmod 4 \text{.}\end{cases}\tag{1.5.3} \end{align}
