Definition 1.1.12. Multiplicative units in \(\Z/n\Z\).

Fix a modulus \(n\text{.}\) An element \([a]_n\in \Z/n\Z\) is a unit (or (multiplicatively) invertible) if there is an element \([b]_n\in \Z/n\Z\) satisfying
\begin{equation*} [a]_n[b]_n=[b]_n[a]_n=[1]_n=1\text{.} \end{equation*}
The element \([b]_n\) in this case is called the multiplicative inverse of \([a]_n\text{,}\) denoted \([a]_n^{-1}\text{.}\) The set of all units of \(\Z/n\Z\) is denoted \((\Z/n\Z)^*\text{:}\) i.e.,
\begin{equation} (\Z/n\Z)^*=\{[a]_n\in \Z/n\Z\mid [a]_n[b]_n=[1]_n \text{ for some } [b]_n\in \Z/n\Z\}\text{.}\tag{1.1.7} \end{equation}
