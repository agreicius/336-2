Example 1.1.13. Modulus \(n=5\).

Compute (a) and addition table for \(\Z/5\Z\text{,}\) and (b) a multiplication table for \(\Z/5\Z\text{.}\)
🔗
From your multiplication table, determine \((\Z/5\Z)^*\text{.}\)
🔗
Solution.
You should do this by hand on your own. We are too lazy to write up the tables here, but will show you how to do this in Sage. See Sage examples for more details about using Sage.
🔗
Looking at the multiplication table for \(\Z/5\Z\text{,}\) we see that with the exception of \(0\text{,}\) all elements \(r\in \Z/5\Z\) have an element \(s\in \Z/5\Z\) satisfying \(r\cdot s=1\text{.}\) Thus \((\Z/5\Z)^*=\Z/5\Z-\{0\}\text{.}\)
