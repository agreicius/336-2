Definition 1.1.9. Ring of integers modulo \(n\).

Let \(n\) be a positive integer. The set \(\Z/n\Z\) together with the modular addition operation
\begin{align*} [a]_n+[b]_n \amp =[a+b]_n \end{align*}
and modular multiplication operations
\begin{align*} [a]_n\cdot [b]_n \amp =[ab]_n \end{align*}
is called the ring of integers modulo \(n\).
🔗
When the context is clear, we will write \(0=[0]_n\) and \(1=[1]_n\text{.}\) Additionally, for any \(x=[a]_n\in \Z/n\Z\text{,}\) we will write \(-x=[-a]_n\text{.}\)
