    Section 1.12 Gaussian integers

    Objectives

    1. Define the Gaussian integers and relate complex norm to number theoretic questions.
      🔗
      🔗
    🔗
    We will define the Gaussian integers as the subset of \(\C\) consisting of complex numbers whose real and imaginary components are integers. This set is an example of a more general construction called a subring of \(\C\text{.}\)
    🔗

    Definition 1.12.1. Gaussian integers.

    The Gaussian integers is the subset \(\Z[i]\) of \(\C\) defined as
    \begin{align} \Z[i] \amp = \{a+ib\in \C\mid a,b\in \Z\} \text{.}\tag{1.12.1} \end{align}
    🔗
    🔗
    One particularly useful property of the Gaussian integers is that they are closed under complex addition, subtraction, and multiplication. This means that \(\Z[i]\) can be considered as a number system (or ring) of its own. Furthermore, since the complex operations can be viewed as extensions of the corresponding real operations, the addition and multiplication operations of \(\Z[i]\) can be viewed as extensions of integer addition and multiplication. In this regard, the Gaussian integers are an example of a more general construction called a subring of \(\C\text{.}\)
    🔗

    Definition 1.12.2. Subring of \(\C\).

    A subring of \(\C\) is a subset \(R\subseteq \C\) containing \(0\) and \(1\) that satisfies the following properties.
    1. Closed under addition/subtraction.
      If \(\alpha,\beta\in R\text{,}\) then \(\alpha+\beta\in R\) and \(\alpha-\beta\in R\text{.}\)
      🔗
      🔗
    2. Closed under multiplication.
      If \(\alpha, \beta\in R\text{,}\) then \(\alpha\beta\in R\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proposition 1.12.3. \(\Z[i]\) is a subring.

    The Gaussian integers form a subring of \(\C\text{.}\)
    🔗
    🔗
    In studying the arithmetic of \(\Z[i]\text{,}\) it turns out that the modulus function \(\abs{a+bi}=\sqrt{a^2+b^2}\) is an invaluable tool. In the current context, it is more convenient to work with \(\abs{a+bi}^2=a^2+b^2\text{,}\) since this is an integer. Accordingly, we give this slight variation of the modulus function a special name.
    🔗

    Definition 1.12.4. Gaussian norm.

    The norm function on \(\Z[i]\) is the function \(N\colon \Z[i]\rightarrow \Z_{\geq 0}\) defined as \(N(\alpha)=\abs{\alpha}^2\text{.}\) Equivalently, if \(\alpha=a+ib\text{,}\) we have
    \begin{align} N(\alpha) \amp = \alpha\cdot \overline{\alpha}=a^2+b^2\text{.}\tag{1.12.2} \end{align}
    🔗
    🔗

    Definition 1.12.5. Units of subrings.

    Let \(R\) be a subring of \(\C\text{.}\) An element \(\alpha\in R\) is a unit (or is invertible) if there exists a \(\beta\in R\) satisfying \(\alpha\beta=1\text{.}\) We denote by \(R^*\) the set of all units in \(R\text{.}\)
    🔗
    🔗

    Remark 1.12.6. Units in \(R\).

    It is crucial to understand that the notion of invertibility in \(R\) is relative to the subring \(R\) in question. Consider the element \(\alpha=1+i\in (\Z[i])^*\text{,}\) for example. Like any nonzero element of \(\C\text{,}\) it has a multiplicative inverse \(\alpha^{-1}\) satisfying \(\alpha\alpha^{-1}=1\text{.}\) However, we have
    \begin{equation*} \alpha^{-1}=\frac{1}{2}-\frac{1}{2}i\not\in \Z[i]\text{.} \end{equation*}
    But is it possible that there is some other element \(\beta\in \Z[i]\) satisfying \(\alpha\beta=1\text{?}\) No! This is a consequence of the fact that inverses in \(\C\) are unique: \(\alpha\alpha^{-1}=\alpha\beta=1\) implies \(\alpha^{-1}=\beta\text{.}\) Thus \(\alpha^{-1}\) not being an element of \(\Z[i]\) is sufficient to conclude that \(\alpha\) has no inverse in the subring \(\Z[i]\text{,}\) and thus that \(\alpha\notin (\Z[i])^*\text{.}\)
    🔗
    🔗
    As a first example of the utility of the norm function, we give an elegant proof of the fact that
    \begin{equation*} (\Z[i])^*=\{1,-1,i,-i\}\text{.} \end{equation*}
    🔗

    Proposition 1.12.7. Units of \(\Z[i]\).

    We have
    \begin{align} (Z[i])^* \amp =\{\alpha\in \Z[i]\mid N(\alpha)=1\}=\{1,-1,i,-i\}\text{.}\tag{1.12.3} \end{align}
    🔗
    🔗

    Proof.

    To prove the first set equality, we must show that \(\alpha\) is a unit if and only if \(N(\alpha)=1\text{.}\)
    🔗
    First assume \(\alpha\in \Z[i]\text{.}\) By definition, there is an element \(\beta\in \Z[i]\) such that \(\alpha\beta=1\text{.}\) Applying \(N\) to both sides of this equation, we see that \(N(\alpha)N(\beta)=N(1)=1\text{.}\) Since \(N(\alpha)\) and \(N(\beta)\) are integers, we conclude that \(N(\alpha)\mid 1\) in \(\Z\text{,}\) and hence that \(N(\alpha)=\pm 1\text{.}\) Since \(N(\alpha)\geq 0\text{,}\) we conclude that \(N(\alpha)=1\text{.}\)
    🔗
    Now assume \(N(\alpha)=1\text{.}\) By definition this means \(\alpha\overline{\alpha}=1\text{.}\) Since \(\overline{\alpha}\in \Z[i]\text{,}\) we conclude that \(\alpha\in (\Z[i])^*\text{.}\)
    🔗
    We now know that \(\alpha=a+bi\in (\Z[i])^*\) if and only if \(N(\alpha)=a^2+b^2=1\text{.}\) Since \(a\) and \(b\) are integers, it is easy to see that we must have \((a,b)\in \{(\pm 1,0),(0,\pm 1)\}\text{,}\) and hence that \((\Z[i])^*=\{1,-1,i,-i\}\text{,}\) as claimed.
    🔗
    🔗
    The modulus function will also be useful in our analysis of the question of which integers can be expressed as a sum of squares.
    🔗

    Proposition 1.12.8. Sum of squares.

    Let \(n\) be a positive integer. The following statements are equivalent.
    1. \(n=a^2+b^2\) for some integers \(a\) and \(b\text{.}\)
      🔗
      🔗
    2. \(n\) factors as \(\alpha\overline{\alpha}\) for some \(\alpha\in \Z[i]\text{.}\)
      🔗
      🔗
    3. \(n=N(\alpha)\) for some \(\alpha\in \Z[i]\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Corollary 1.12.9. Products of sums of squares.

    If \(m\) and \(n\) are integers that can be written as a sum of two squares of integers, then \(mn\) can be written as a sum of two squares of integers.
    🔗
    🔗

    Proof.

    Assume \(m=a^2+b^2\) and \(n=c^2+d^2\text{.}\) Letting \(\alpha=a+bi\) and \(\beta=c+di\text{,}\) we have
    \begin{align*} m \amp =N(\alpha) \amp n=N(\beta) \end{align*}
    and hence
    \begin{align*} mn \amp =N(\alpha)N(\beta)=N(\alpha\beta)\text{.} \end{align*}
    It follows from Proposition 1.12.8 that \(mn\) is a sum of squares. In fact, writing
    \begin{equation*} \alpha\beta=(ac-bd)+(ad+bc)i\text{,} \end{equation*}
    we see that
    \begin{equation*} mn=(ac-bd)^2+(ad+bc)^2\text{.} \end{equation*}
    🔗
    🔗

    Definition 1.12.10. Divisibility in subrings.

    Let \(R\) be a subring of \(\C\text{.}\) Given \(\alpha,\beta\in R\text{,}\) we say that \(\alpha\) divides \(\beta\) in \(R\text{,}\) denoted \(\alpha\mid \beta\text{,}\) if there exists a \(\gamma\in R\) such that \(\beta=\alpha\gamma\text{.}\) We write \(\alpha\nmid \beta\) if \(\alpha\) does not divide \(\beta\text{.}\)
    🔗
    We say that \(\alpha\) is an associate of \(\beta\) if \(\alpha=u \beta\) for some unit \(u\in R^*\text{.}\)
    🔗
    🔗

    Example 1.12.11. Divisibility in \(\Z[i]\).

    For the given \(\alpha, \beta\in \Z[i]\) determine whether \(\alpha\mid \beta\) and whether \(\beta\mid \alpha\text{.}\)
    1. \(\alpha=1+2i\text{,}\) \(\beta=3+i\)
      🔗
      🔗
    2. \(\alpha=1+2i\text{,}\) \(\beta=2-i\)
      🔗
      🔗
    🔗
    Solution.
    One way of answering this type of question is to use some logic similar to Remark 1.12.6, zooming out a bit to the larger number system \(\C\text{.}\) In general, for nonzero elements \(\alpha, \beta\in \C\text{,}\) we have \(\beta=\gamma \alpha\) for if and only if \(\gamma=\alpha^{-1}\beta\text{.}\) Thus we have \(\alpha\mid \beta\) in a subring \(R\) if and only if \(\alpha^{-1}\beta\in R\text{.}\)
    🔗
    Working in \(\Z[i]\text{,}\) we can also use the norm function to give a necessary (though not necessarily sufficient condition) for divisibility: namely, if \(\alpha\mid \beta\text{,}\) then we have \(\beta=\alpha\gamma\) for some \(\gamma\in \Z[i]\text{,}\) in which case \(N(\beta)=N(\alpha)N(\gamma)\) and \(N(\alpha)\mid N(\beta)\text{.}\)
    🔗
    We illustrate both techniques in the two examples.
    1. Since \(N(\alpha)=5\) and \(N(\beta)=10\text{,}\) we see that \(\beta\nmid \alpha\text{,}\) but possibly \(\alpha\mid \beta\text{.}\) To decide the latter question, we compute
      \begin{equation*} \alpha^{-1}\beta=(1/5-(2/5)i)(3+i)=1-i\in \Z[i]\text{.} \end{equation*}
      Thus \(\alpha\mid \beta\text{.}\)
      🔗
      🔗
    2. In this case we have \(N(\alpha)=N(\beta)=5\text{.}\) Some arithmetic similar to the above shows that \(\alpha=-i\beta\) and \(\beta=-i\alpha\text{.}\) We conclude that \(\alpha\mid \beta\) and \(\beta\mid \alpha\text{.}\)
      🔗
      🔗
    🔗
    🔗
    🔗

    Proposition 1.12.12.

    Let \(\alpha\) and \(\beta\) be nonzero elements of \(R\text{.}\) The following statements are equivalent.
    1. \(\alpha\) is associate to \(\beta\text{.}\)
      🔗
      🔗
    2. \(\beta\) is associate to \(\alpha\text{.}\)
      🔗
      🔗
    3. \(\alpha\mid \beta\) and \(\beta\mid \alpha\) in \(R\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Proof.

    Implication: (i) implies (ii). If \(\alpha\) is associate to \(\beta\text{,}\) then there is a unit \(u\in R^*\) such that \(\alpha=u \beta\text{.}\) Since \(u\in R^*\text{,}\) we have \(u^{-1}\in R^*\) and \(\beta=u^{-1}\alpha\text{.}\) Thus \(\beta\) is associate to \(\alpha\text{.}\)
    🔗
    Implication: (ii) implies (i). If \(\beta\) is associate to \(\alpha\text{,}\) we have \(\beta=u \alpha\) with \(u\in R^*\text{.}\) In particular, this means \(\alpha\mid \beta\text{.}\) Since \(u\in R^*\text{,}\) there is an element \(u^{-1}\in R\) such that \(uu^{-1}=1\text{.}\) It follows that \(\alpha=u^{-1}\beta\) and \(\beta\mid \alpha\text{.}\)
    🔗
    Implication: (iii) implies (i). If \(\alpha \mid \beta\) and \(\beta\mid \alpha\text{,}\) we have
    \begin{align*} \beta \amp = \gamma \alpha\\ \alpha \amp = \delta \beta \end{align*}
    for some elements \(\gamma, \delta\in R\text{.}\) But then we have
    \begin{equation*} \alpha=\delta \beta=(\delta\gamma) \alpha\text{.} \end{equation*}
    Since \(\alpha\) is nonzero, it has an inverse \(\alha^{-1}\in \C\) (not necessarily an element of \(R\)), that allows us to multiplicatively cancel \(\alpha\) to obtain \(1=\delta\gamma\text{.}\) It follows that \(\delta\in R^*\) and \(\gamma\in R^*\text{,}\) and hence that \(\alpha\) is a associate to \(\beta\text{.}\)
