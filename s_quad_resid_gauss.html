    Section 1.6 Quadratic residues: Gauss’s lemma

    Objectives

    1. State and prove Gauss’s lemma for computing the Legendre symbol.
      🔗
      🔗
    2. Derive a formula for \(\qr{2}{p}\) using Gauss’s lemma.
      🔗
      🔗
    🔗

    Theorem 1.6.1. Gauss’s lemma for quadratic residues.

    Let \(p\) be an odd prime, and let \(a\) be an integer with \(p\nmid a\text{.}\) Consider the set
    \begin{equation*} X=\{[ka]_p\mid 1\leq k\leq (p-1)/2\}\subseteq \F_p^*\text{,} \end{equation*}
    and let \(n\) be the number of elements of \(X\) that lie in the set \(\{[(p+1)/2]_p, \dots, [p-1]_p\}\text{.}\) Equivalently, \(n\) is the number of integers \(k\in \{1,2,\dots, (p-1)/2\}\) such that the least residue of \(ka\) is greater than \((p-1)/2\text{.}\)
    \begin{align} \qr{a}{p} \amp = (-1)^n \text{.}\tag{1.6.1} \end{align}
    🔗
    🔗

    Proof.

    Let \(r_1,r_2,\dots, r_n\) be the elements \([k a]_p\) of \(X\) that lie in the set \(\{[(p+1)/2]_p, \dots, [p-1]_p\}\text{,}\) and let \(s_1,s_2,\dots, s_m,\) be the remaining elements of \(X\text{.}\) Note that these elements are all distinct, since \([i]_p[a]_p=[j]_p[a]_p\) implies \([i]_p=[j]_p\text{,}\) by cancellation. In particular, we have \(n+m=(p-1)/2\text{.}\)
    🔗
    Next, we claim that for each \(r_i\text{,}\) we have \(-r_i\in \{[1]_p, [2]_p, \dots, [(p-1)/2]_p\}\text{.}\) This is because for all \(i\text{,}\) \(r_i=[b]_p\) for some \(p/2< b\leq p-1\text{,}\) in which case \(-r_i=[-b]_p=[p-b]_p\text{,}\) and \(1\leq p-b < p/2\text{.}\)
    🔗
    In fact, we claim further that
    \begin{equation*} \{-r_1,-r_2,\dots,-r_n,s_1,s_2,\dots, s_m\}=\{[1]_p,\dots, [(p-1)/2]_p\}\text{.} \end{equation*}
    From the discussion above, we know that the inclusion \(\subseteq\) holds. To see that this is in fact an equality, we will show that
    \begin{align} \abs{-r_1,-r_2,\dots, -r_n,s_1,s_2,\dots, s_m} \amp \tag{1.6.2}\\ \amp = \abs{\{[1]_p,\dots, [(p-1)/2]_p\}}=(p-1)/2 \text{.}\tag{1.6.3} \end{align}
    Since \(n+m=(p-1)/2\text{,}\) it suffices to show that the elements \(-r_i\) and \(s_j\) are pairwise distinct. Since \(r_i\ne r_j\) and \(s_i\ne s_j\) for all \(i\ne j\text{,}\) it suffices to show that \(-r_i\ne s_j\) for all \(i,j\text{.}\) We prove this by contradiction. If \(-r_i=s_j\text{,}\) where \(r_i=[-\ell a]_p\) and \(s_j=[k a]_p\) for some \(1\leq \ell, k\leq (p-1)/2\text{,}\) then we would have
    \begin{equation*} [k]_p[a]_p+[\ell]_p[a]_p=([k+l]_p)[a]_p=0\text{.} \end{equation*}
    Since \([a]_p\) is a unit, we conclude that \([k+\ell]_p=0\text{.}\) This is a contradiction since \(1\leq k+\ell\leq p-1\text{,}\) and thus \(p\nmid k+\ell\text{.}\)
    🔗
    Now that we know that \(\{-r_1,-r_2,\dots,-r_n,s_1,s_2,\dots, s_m\}=\{[1]_p,\dots, [(p-1)/2]_p\}\text{,}\) we conclude that
    \begin{align*} \prod_{i=1}^n(-r_i)\prod_{j=1}^ms_j \amp = \prod_{i=1}^{(p-1)/2}[i]_p \\ \amp= [((p-1)/2)!]_p\text{.} \end{align*}
    On the other hand, since the union of the \(r_i\) and \(s_j\) is the set of all multiples \([ka]_p\) with \(1\leq k\leq (p-1)/2\text{,}\) we have
    \begin{align} \prod_{i=1}^n(-r_i)\prod_{j=1}^ms_j\amp = [-1]_p^n \prod_{k=1}^{(p-1)/2}[ka]_p \tag{1.6.4}\\ \amp = [-1]_p^n [a]_p^{(p-1)/2}[((p-1)/2)!]_p\text{.}\tag{1.6.5} \end{align}
    We conclude that
    \begin{align} [((p-1)/2)!]_p \amp = [(-1)]_p^n [a]_p^{(p-1)/2}[((p-1)/2)!]_p \text{,}\tag{1.6.6} \end{align}
    and thus \(1=(-1)^n[a]_p^{(p-1)/2}\text{,}\) since \([((p-1)/2)!]_p\) is a unit. Multiplying both sides by \([-1]_p^n\text{,}\) we conclude that \([a]_p^{(p-1)/2}=[-1]_p^n\text{.}\) Euler’s criterion (Theorem 1.5.7) now implies \(\qr{a}{p}\equiv (-1)^n \pmod p\text{.}\) Since \(\qr{a}{p}\in \{-1,1\}\) and \(p> 2\text{,}\) it follows that \(\qr{a}{p}=(-1)^n\text{.}\)
    🔗
    🔗

    Example 1.6.4. Gauss’s lemma: computing \(\qr{5}{11}\).

    Compute \(\qr{5}{11}\) using Gauss’s lemma and confirm your answer by brute force.
    🔗
    Solution.
    We work within \(\F_{11}\) and drop the brackets. The multiples \(k\cdot 5\in \F_{11}\) for \(1\leq k\leq 5\) are \(5,10,4,9,3\text{.}\) Two of these lie in the set \(\{6,7,8,9,10\}\text{.}\) Thus \(\qr{5}{11}=(-1)^2=1\text{.}\) (See xref{fig_gauss_5squaremod11}). Indeed, by inspection we find that \(7^2=49=5\) in \(\F_{11}\text{.}\)

    Corollary 1.6.6. Formula for \(\qr{2}{p}\).

    Let \(p\) be an odd prime. We have
    \begin{equation} \qr{2}{p}=\begin{cases} 1 \amp \text{if } p\equiv \pm 1 \pmod 8 \\ -1 \amp \text{if } p\equiv \pm 3 \pmod 8 \text{.}\end{cases}\tag{1.6.7} \end{equation}
    🔗
    🔗

    Proof.

    Let
    \begin{align} X \amp =\{[1]_p, \dots [(p-1)/2]_p\}\tag{1.6.8}\\ Y \amp =\{[(p+1)/2]_p,\dots ,[p-1]_p\}\text{.}\tag{1.6.9} \end{align}
    Using Theorem 1.6.1, we have
    \begin{equation*} \qr{2}{p}=(-1)^n\text{,} \end{equation*}
    where \(n\) is the number of multiples of \([2k]_p\text{,}\) \(1\leq k\leq (p-1)/2\text{,}\) that lie in \(Y\text{.}\)
    🔗
    We treat two separate cases: \(p\equiv 1\pmod 4\) and \(p\equiv -1\pmod 4\text{.}\)
    🔗

    Case: \(p\equiv 1 \pmod 4\).

    Assume \(p\equiv 1\pmod 4\text{,}\) or equivalently, \(p-1=4k\) for some integer \(k\text{.}\) In this case we have \((p-1)/2=2k\text{,}\) which is divisible by \(2\text{.}\) It follows that there are exactly \((p-1)/4\) multiples of \([2]_p\) in \(X\) and \(Y\text{,}\) and thus that
    \begin{equation*} \qr{2}{p}=(-1)^{(p-1)/4}\text{.} \end{equation*}
    Since \((p-1)/4=k\in \Z\text{,}\) we see that this exponent is even if and only if \(2\mid k\text{,}\) and it is easy to see that this is true if and only if \(p\equiv 1\pmod 8\text{.}\) Since \(p\equiv 1\pmod 4\) implies \(p\equiv 1 \pmod 8\) or \(p\equiv 5 \pmod 8\text{,}\) we conclude that in this case we have
    \begin{equation*} \qr{2}{8}=\begin{cases} 1 \amp \text{if } p\equiv 1\pmod 8\\ -1 \amp \text{if } p\equiv 5\pmod 8 \text{.}\end{cases} \end{equation*}
    🔗
    🔗

    Case: \(p\equiv 3\pmod 4\).

    Assume \(p\equiv 3\pmod 4\text{,}\) or equivalently, \(p=4k+3\) for some integer \(k\text{.}\) In this case \((p-3)/2\) is a multiple of \(2\text{,}\) but \((p-1)/2\) is not. It follows that there are \((p-3)/4\) multiples os \([2]_p\) lying in \(X\) and \((p+1)/4\) multiples of \([2]_p\) lying in \(Y\text{.}\) We conclude that
    \begin{equation*} \qr{2}{p}=(-1)^{(p+1)/4}\text{.} \end{equation*}
    Since \((p+1)/4=(k+1)\text{,}\) we see that this exponent is even if and only if \(k\) is odd, which is true if and only if \(p\equiv 7 \pmod 8\text{.}\) Since \(p\equiv 3\pmod 4\) implies \(p\equiv 3 \pmod 8\) or \(p\equiv 7 \pmod 8\text{,}\) we conclude in this case that
    \begin{equation*} \qr{2}{8}=\begin{cases} 1 \amp \text{if } p\equiv 7\pmod 8\\ -1 \amp \text{if } p\equiv 3 \pmod 8 \text{.}\end{cases} \end{equation*}
    🔗
    🔗
    Since our two cases are exhaustive, we combine them to conclude
    \begin{equation*} \qr{2}{8}=\begin{cases} 1 \amp \text{if } p\equiv 1\pmod 8 \\ -1\amp \text{if } p\equiv 5\pmod 8\\ 1 \amp \text{if } p\equiv 7\pmod 8\\ -1\amp \text{if } p\equiv 3\pmod 8 \text{,}\end{cases} \end{equation*}
    or more succinctly,
    \begin{equation*} \qr{2}{8}=\begin{cases} 1 \amp \text{if } p\equiv \pm 1\pmod 8\\ -1\amp \text{if } p\equiv \pm 3\pmod 8 \text{.}\end{cases} \end{equation*}
